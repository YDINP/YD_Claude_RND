# ArcaneCollectors 신규 기능 제안서

**작성일**: 2025-02-04
**목적**: 게임의 장기 플레이 가치와 수익화 잠재력을 높이기 위한 신규 기능 제안

---

## 1. 게임플레이 확장 제안

### 1.1 아레나 (PvP) 시스템
**우선순위**: ⭐⭐⭐⭐⭐

**개요**: 실시간/비동기 PvP 콘텐츠

**기능**:
- **실시간 아레나**: 매칭 기반 1:1 대전
- **수비 파티 설정**: AI가 수비 파티 운영
- **주간 시즌**: 티어 시스템 (브론즈~챔피언)
- **보상**: 아레나 코인 → 전용 상점

**수익화 포인트**:
- 아레나 티켓 충전
- 시즌 패스 (추가 보상)

**예상 구현**:
```javascript
// ArenaSystem.js
class ArenaSystem {
  matchmaking(playerRating) { /* 레이팅 기반 매칭 */ }
  setDefenseParty(partyId) { /* 수비 파티 설정 */ }
  calculateRating(result) { /* ELO 레이팅 계산 */ }
  getSeasonRewards(tier) { /* 시즌 보상 */ }
}
```

---

### 1.2 월드 보스 시스템
**우선순위**: ⭐⭐⭐⭐

**개요**: 서버 전체 협동 보스 레이드

**기능**:
- **실시간 HP 공유**: 전체 유저 데미지 누적
- **기여도 보상**: 데미지 비율에 따른 보상
- **주간 교체**: 매주 다른 보스 등장
- **길드 경쟁**: 길드별 총 데미지 랭킹

**특별 메커닉**:
- 보스 패턴 변화 (HP 50% 이하)
- 약점 성격 시스템 연동
- 버프/디버프 스택 시스템

---

### 1.3 원정대 시스템 (자동 탐험)
**우선순위**: ⭐⭐⭐⭐

**개요**: 방치형 자동 보상 시스템

**기능**:
- **원정 슬롯**: 영웅 파견 (4~12시간)
- **보상 종류**: 골드, 경험치, 재료, 장비
- **희귀 보상**: 낮은 확률로 SSR 획득 가능
- **VIP 혜택**: 슬롯 수 증가

**UI**:
```
[원정 1] 북유럽 설원 - 4시간 - 진행중 (2:30:15)
[원정 2] 동방 신전 - 8시간 - 완료! [수령]
[원정 3] 🔒 VIP 3 필요
```

---

### 1.4 영웅 호감도 시스템
**우선순위**: ⭐⭐⭐

**개요**: 영웅과의 친밀도 시스템

**기능**:
- **호감도 레벨**: 1~10단계
- **스토리 해금**: 레벨별 개인 스토리
- **보너스 스탯**: 호감도에 따른 추가 스탯
- **선물 시스템**: 아이템으로 호감도 상승

**캐릭터 스토리 예시**:
```
[토르 호감도 5 스토리]
"묠니르를 처음 들어올린 날의 이야기를 해줄까?
그때 난 아직 어린 신이었지..."
```

---

### 1.5 도감 시스템
**우선순위**: ⭐⭐⭐

**개요**: 수집 요소 + 보너스 시스템

**기능**:
- **영웅 도감**: 획득한 영웅 기록
- **장비 도감**: 획득한 장비 기록
- **몬스터 도감**: 처치한 적 기록
- **도감 보상**: 완성도에 따른 영구 버프

**도감 보상 예시**:
```
발할라 도감 50% 완성 → ATK +2%
발할라 도감 100% 완성 → ATK +5%, 북유럽 시너지 +10%
```

## 2. 소셜 & 커뮤니티 기능

### 2.1 길드 시스템
**우선순위**: ⭐⭐⭐⭐⭐

**개요**: 길드 기반 협동 콘텐츠

**기능**:
- **길드 생성**: 최대 30명
- **길드 레벨**: 기여도로 레벨업
- **길드 스킬**: 전체 멤버 버프
- **길드 전쟁**: 주간 길드 대항전
- **길드 보스**: 협동 보스 레이드

**길드 스킬 트리**:
```
공격력 +5% → +10% → +15%
방어력 +5% → +10% → +15%
경험치 +10% → +20% → +30%
골드 +10% → +20% → +30%
```

---

### 2.2 실시간 채팅
**우선순위**: ⭐⭐⭐

**개요**: 길드/월드 채팅

**기능**:
- **월드 채팅**: 전체 유저 채팅
- **길드 채팅**: 길드 내 채팅
- **귓속말**: 1:1 채팅
- **이모티콘**: 게임 캐릭터 이모티콘

---

### 2.3 업적 시스템
**우선순위**: ⭐⭐⭐⭐

**개요**: 장기 목표 + 보상 시스템

**카테고리**:
- **전투**: 적 처치, 스테이지 클리어
- **수집**: 영웅, 장비 획득
- **소셜**: 친구, 길드 활동
- **이벤트**: 시즌 업적

**업적 예시**:
```
[신화의 수집가] SSR 영웅 10종 획득 → 보석 500개
[전장의 전설] 10,000 전투 승리 → 칭호 "전설"
[부의 축적] 골드 1,000,000 획득 → 골드 부스트 +10%
```

## 3. 경제 시스템 개선

### 3.1 거래소 시스템
**우선순위**: ⭐⭐⭐

**개요**: 유저 간 아이템 거래

**기능**:
- **장비 거래**: 강화된 장비 판매
- **재료 거래**: 진화/강화 재료
- **가격 시스템**: 유저 설정 가격
- **수수료**: 거래액의 10%

**제한사항**:
- 영웅 거래 불가
- 일일 거래 횟수 제한
- 최소 거래 가격

---

### 3.2 VIP 시스템
**우선순위**: ⭐⭐⭐⭐⭐

**개요**: 과금 유저 혜택

**VIP 레벨별 혜택**:

| VIP | 필요 충전액 | 혜택 |
|-----|------------|------|
| 1 | 5,000원 | 소탕 +10회, 에너지 회복 +10% |
| 2 | 20,000원 | 소탕 +20회, 가챠 확률 +0.5% |
| 3 | 50,000원 | 원정 슬롯 +1, 일일 보석 +10 |
| 4 | 100,000원 | 전용 스킨 1개, 골드 부스트 +20% |
| 5 | 200,000원 | 전용 칭호, 경험치 부스트 +30% |

---

### 3.3 시즌 패스
**우선순위**: ⭐⭐⭐⭐⭐

**개요**: 월간 프리미엄 패스

**구조**:
- **무료 트랙**: 기본 보상
- **프리미엄 트랙**: 추가 보상 (유료)
- **레벨**: 1~50 (일일 퀘스트로 경험치)

**보상 예시**:
```
Lv.10: 무료(골드 5000) / 프리미엄(보석 50)
Lv.25: 무료(R 영웅) / 프리미엄(SR 영웅)
Lv.50: 무료(SR 장비) / 프리미엄(SSR 선택권)
```

---

## 4. 이벤트 시스템

### 4.1 출석 이벤트
**우선순위**: ⭐⭐⭐⭐⭐

**기능**:
- **7일 출석**: 매주 리셋
- **누적 출석**: 월간 누적 보상
- **연속 출석**: 연속 로그인 보너스

**7일 출석 보상**:
```
1일: 골드 10,000
2일: 에너지 포션 x5
3일: 소탕권 x10
4일: R 영웅 소환권
5일: 보석 100
6일: SR 장비 상자
7일: SR 영웅 선택권
```

---

### 4.2 기간 한정 이벤트
**우선순위**: ⭐⭐⭐⭐

**유형**:
- **신규 영웅 픽업**: 2주간 특정 영웅 확률 UP
- **교단 이벤트**: 특정 교단 보너스 기간
- **레이드 이벤트**: 특별 보스 등장
- **명절 이벤트**: 설날, 추석 등 특별 보상

---

### 4.3 복귀 유저 이벤트
**우선순위**: ⭐⭐⭐

**조건**: 7일 이상 미접속

**보상**:
- 복귀 로그인 보너스 (7일)
- 따라잡기 패키지 할인
- 경험치/골드 2배 (3일간)
