**AI 선택 로직:**
```javascript
class AIDecisionMaker {
  selectAction(character, battleState) {
    const options = [
      { action: 'attack', priority: 1 },
      { action: 'heal', priority: character.canHeal ? 5 : 0 },
      { action: 'buff', priority: 3 },
      { action: 'skill', priority: 2 }
    ];

    const best = options
      .filter(o => o.priority > 0)
      .sort((a, b) => b.priority - a.priority)[0];

    return best.action;
  }
}
```

**완료 조건:**
