# ArcaneCollectors 캐릭터 디자인 가이드

> **스타일**: 트릭컬 리바이브 (Trickle Revive) - 귀엽고 발칙한 SD 스타일

## 디자인 철학

- **비율**: 2.5~3등신의 슈퍼 디포르메
- **표정**: 과장되고 생동감 넘치는 표현
- **색상**: 밝고 선명한 팔레트
- **라인**: 동글동글하고 부드러운 외형선
- **성격**: 장난스럽고 익살맞은 면모 강조

## SSR 캐릭터 (5성)

### 1. 프레야 (Freya)
**바람 속성 / 공격형**

#### 기본 정보
- 이름: Freya (프레야)
- 등급: SSR (5성)
- 속성: 바람
- 역할: 공격
- 컨셉: 자신감 넘치는 발랄한 전쟁과 사랑의 여신

#### 외형 설명 (2.5등신)
- **체형**: 작고 통통한 2.5등신 비율, 짧은 팔다리가 귀여움 강조
- **헤어스타일**: 허리까지 내려오는 금발 웨이브 헤어, 양 옆에 작은 깃털 장식
- **눈**: 크고 반짝이는 에메랄드빛 녹색 눈, 자신감 있는 윙크 표정이 시그니처
- **의상**:
  - 흰색과 금색의 미니 갑옷 드레스
  - 어깨에 작은 날개 장식
  - 허벅지까지 오는 흰색 부츠
  - 치마는 짧고 펄럭이는 느낌
- **악세서리**:
  - 황금 날개 티아라
  - 목에 두른 붉은 리본
  - 손에 든 미니어처 검 (본인보다 작음)

#### 성격 (트릭컬 스타일)
- **기본 성격**: 당당하고 자신감 넘치며, 전투를 즐기는 활발한 성격
- **발칙한 면**: "제가 최고예요~♪" 하며 자랑하다가 넘어지는 덜렁이
- **버릇/말투**: "에헤헤~", "저 완벽하거든요?" 끝에 하트 이모티콘 붙이듯 말함
- **귀여운 갭**: 무서운 걸 보면 뒤에 숨으면서도 "전혀 안 무서운데요?" 허세

#### 시그니처 포즈
양손을 허리에 올리고 턱을 들어 자신만만하게 웃는 포즈, 한쪽 눈을 윙크하며 검을 비스듬히 어깨에 걸침

#### 컬러 팔레트
```
주색상:
- 골드: #FFD700
- 화이트: #FFFFFF
- 스카이블루: #87CEEB

보조색:
- 에메랄드 그린: #50C878 (눈동자)
- 루비 레드: #E0115F (리본)
```

### 2. 아마테라스 (Amaterasu)
**빛 속성 / 지원형**

#### 기본 정보
- 이름: Amaterasu (아마테라스)
- 등급: SSR (5성)
- 속성: 빛
- 역할: 지원
- 컨셉: 세상 귀찮지만 어쩔 수 없이 빛나는 태양신

#### 외형 설명 (2.5등신)
- **체형**: 날씬한 2.5등신, 축 늘어진 자세가 기본 (귀차니즘 표현)
- **헤어스타일**: 허리까지 오는 새하얀 은발, 뒤로 묶었지만 삐져나온 머리카락들
- **눈**: 황금빛 반달 눈, 항상 반쯤 감은 졸린 표정
- **의상**:
  - 빨강-흰색의 미코(무녀) 의상을 트릭컬식으로 단순화
  - 큰 리본이 뒤로 처져있음
  - 소매가 길어서 손이 잘 안 보임
  - 맨발에 흰색 타비 양말만 착용 (귀찮아서 신발 안 신음)
- **악세서리**:
  - 머리 양옆의 태양 문양 헤어핀
  - 등 뒤의 작은 원형 후광 (OFF 되어있다가 일할 때만 켜짐)
  - 허리춤의 거울 모양 부적

#### 성격 (트릭컬 스타일)
- **기본 성격**: 귀찮음을 극도로 싫어하지만 책임감은 있는 츤데레형 태양신
- **발칙한 면**: "하아... 일하기 싫은데..." 하다가 막상 일 시작하면 완벽주의
- **버릇/말투**: "에휴...", "귀찮네...", 한숨을 자주 쉼, 느릿느릿 말함
- **귀여운 갭**: 칭찬받으면 "별로 안 기쁜데..." 하면서 후광이 환하게 빛남

#### 시그니처 포즈
한 손으로 하품하며 눈물 닦는 포즈, 다른 손은 축 늘어뜨리고 약간 구부정한 자세

#### 컬러 팔레트
```
주색상:
- 화이트: #FFFFFF
- 크림슨 레드: #DC143C
- 골드: #FFD700

보조색:
- 실버: #C0C0C0 (머리카락)
- 선샤인 옐로우: #FFDB58 (후광)
```

### 3. 제우스 (Zeus)
**불 속성 / 공격형**

#### 기본 정보
- 이름: Zeus (제우스)
- 등급: SSR (5성)
- 속성: 불
- 역할: 공격
- 컨셉: 카리스마 넘치지만 바람둥이인 신들의 왕

#### 외형 설명 (3등신)
- **체형**: 탄탄한 3등신, 작지만 위압감 있는 비율
- **헤어스타일**: 흰색 웨이브 머리, 옆으로 넘긴 앞머리가 바람에 휘날림
- **눈**: 샤프한 번개 문양이 들어간 푸른 눈, 자신감 넘치는 눈빛
- **의상**:
  - 흰색 토가를 트릭컬식으로 단순화 (한쪽 어깨 노출)
  - 금색 벨트와 어깨 보호대
  - 보라색 망토 (짧고 휘날림)
  - 맨발에 금색 샌들
- **악세서리**:
  - 황금 월계관
  - 팔에 찬 금색 팔찌
  - 손에 든 미니 번개창 (뽀짝한 사이즈)

#### 성격 (트릭컬 스타일)
- **기본 성격**: 당당하고 카리스마 넘치지만 여자만 보면 헤롱헤롱
- **발칙한 면**: "그대, 나와 함께..." 작업 거는데 번개창으로 자기 발 찌르는 실수
- **버릇/말투**: "호오호오~", 폼 잡다가 삐끗하는 게 일상, "왕의 위엄을!" 외치며 포즈
- **귀여운 갭**: 아내한테 혼날 때는 쪼그라들어서 "죄송합니다..." 모드

#### 시그니처 포즈
한 손은 허리에, 다른 손은 번개창을 하늘로 치켜들며 윙크하는 포즈 (별이 반짝)

#### 컬러 팔레트
```
주색상:
- 화이트: #FFFFFF
- 골드: #FFD700
- 일렉트릭 블루: #7DF9FF

보조색:
- 퍼플: #800080 (망토)
- 스카이 블루: #87CEEB (눈)
```

### 4. 토르 (Thor)
**물 속성 / 탱커형**

#### 기본 정보
- 이름: Thor (토르)
- 등급: SSR (5성)
- 속성: 물
- 역할: 탱커
- 컨셉: 단순하지만 강력한 천둥의 신

#### 외형 설명 (3등신)
- **체형**: 뭉툭하고 든든한 3등신, 어깨가 넓고 팔다리가 굵음
- **헤어스타일**: 빨간색 단발 헤어, 산발되어 삐져나온 머리카락들
- **눈**: 작고 동그란 파란 눈, 순수하고 단순한 표정
- **의상**:
  - 검은색 민소매 갑옷 (배가 약간 나옴)
  - 빨간 망토 (짧고 두꺼움)
  - 금속 벨트에 망치 거치대
  - 무릎까지 오는 갈색 바지와 부츠
- **악세서리**:
  - 양손에 낀 가죽 장갑
  - 팔뚝의 금속 암밴드
  - 묠니르 (Mjolnir) 망치 - 본인 키의 절반 크기

#### 성격 (트릭컬 스타일)
- **기본 성격**: 단순 무식하지만 정의롭고 힘이 센 전사
- **발칙한 면**: "토르 강해!" "토르 부숴!" 3인칭 화법으로 자기 자랑
- **버릇/말투**: "으랏차!", "토르 배고파~", 짧고 단순한 문장만 사용
- **귀여운 갭**: 작은 동물 보면 "조그맣다..." 하며 조심조심 쓰다듬음

#### 시그니처 포즈
망치를 양손으로 들어올리며 "으랏차!" 하는 포즈, 발을 어깨 너비로 벌리고 씩씩함 표현

#### 컬러 팔레트
```
주색상:
- 블랙: #000000
- 크림슨 레드: #DC143C
- 실버: #C0C0C0

보조색:
- 브라운: #8B4513 (장갑, 부츠)
- 오션 블루: #4F97A3 (눈)
```

### 5. 이자나미 (Izanami)
**암흑 속성 / 디버퍼**

#### 기본 정보
- 이름: Izanami (이자나미)
- 등급: SSR (5성)
- 속성: 암흑
- 역할: 디버퍼
- 컨셉: 차갑고 냉정한 저승의 여왕

#### 외형 설명 (2.5등신)
- **체형**: 가늘고 우아한 2.5등신, 거의 떠있는 듯한 자세
- **헤어스타일**: 허리까지 내려오는 검은 생머리, 끝부분만 보라빛으로 그라데이션
- **눈**: 날카로운 보라색 눈, 차갑고 무표정한 시선
- **의상**:
  - 검은색 기모노를 트릭컬식으로 단순화
  - 긴 소매가 손을 덮음
  - 보라색 오비 (허리띠)
  - 검은 타비와 게타 (나막신)
  - 치마 끝자락이 안개처럼 흩어짐
- **악세서리**:
  - 머리에 꽂힌 검은 비녀
  - 손목의 흰색 염주
  - 주변을 떠다니는 작은 혼령들 (파란 불꽃)

#### 성격 (트릭컬 스타일)
- **기본 성격**: 냉정하고 감정 표현이 적지만 속은 따뜻함
- **발칙한 면**: "별로 관심 없어." 하면서 주변 혼령들한테는 "배고프니?" 챙겨줌
- **버릇/말투**: "...그래.", "상관없어.", 짧고 무뚝뚝하게 끊어 말함
- **귀여운 갭**: 귀여운 걸 보면 시선을 못 떼지만 "그냥 봤을 뿐." 하며 고개 돌림

#### 시그니처 포즈
두 손을 기모노 소매 안에 넣고 고개를 약간 숙인 채 옆을 바라보는 포즈, 주변에 혼령 2~3개

#### 컬러 팔레트
```
주색상:
- 블랙: #000000
- 딥 퍼플: #4B0082
- 화이트: #F5F5F5

보조색:
- 고스트 블루: #87CEEB (혼령)
- 다크 바이올렛: #9400D3 (눈)
```

### 공통 규칙
1. **눈은 크게**: 얼굴의 1/3 비율, 감정 표현 극대화
2. **손발은 작게**: 뽀짝하고 귀여운 느낌
3. **머리는 크게**: 몸의 40% 이상 비율
4. **표정은 과장**: 기쁨/슬픔/화남 모두 150% 표현
5. **색상은 선명**: 채도 높은 원색 사용

### 등급별 차별화
- **SSR**: 가장 화려한 색상과 복잡한 악세서리, 후광/번개/불꽃 등 이펙트
