> **장르**: 수집형 턴제 RPG
> **플랫폼**: 웹/모바일 (HTML5)
> **엔진**: Phaser 3
> **버전**: 3.0

---

## 1. 게임 컨셉

### 1.1 핵심 콘셉트
**"균열의 시대 - 신화 세계의 충돌"**

세계수 이그드라실에 균열이 생기며 북유럽, 일본, 그리스 신화 세계가 하나로 뒤섞였다. 플레이어는 각 신화권의 영웅들을 모아 균열을 막아야 한다.

### 1.2 핵심 재미 요소
| 요소 | 설명 |
|-----|------|
| **수집** | 신화 속 신과 영웅을 가챠로 수집 |
| **육성** | 레벨업, 진화, 장비 강화 |
| **전략** | 교단/속성 시너지를 활용한 팀 구성 |
| **스토리** | 신화 세계관 기반 메인 스토리 |

### 1.3 타겟 유저
- 서브컬처 게임 선호 유저
- 수집형 RPG 애호가
- 신화/판타지 팬

## 2. 세계관

### 2.1 배경 스토리
```
어느 날, 세계를 연결하던 세계수 이그드라실에 균열이 생기며
각기 다른 신화 세계가 하나로 뒤섞이기 시작했다.

북유럽의 발할라, 일본의 타카마가하라, 그리스의 올림푸스...
신들은 자신의 세계를 지키기 위해 전사들을 소환하고,
플레이어는 이 혼돈 속에서 균열을 막을 영웅들을 모아야 한다.
```

### 2.2 5대 교단 (Cult)

| 교단 | 기원 | 대표 속성 | 특징 | 테마 색상 |
|-----|------|---------|------|----------|
| **발할라** | 북유럽 전사 | Wind | 공격적, 전사 중심 | #4A90D9 (파랑) |
| **타카마가하라** | 일본 신화 | Light | 빠른 속도, 크리티컬 | #FFD700 (황금) |
| **올림푸스** | 그리스 신화 | Fire | 높은 공격력 | #FF6B35 (주황) |
| **아스가르드** | 북유럽 신 | Water | 높은 방어력/HP | #5DADE2 (청록) |
| **요미** | 일본 저승 | Dark | 치명타 데미지 | #8E44AD (보라) |

## 3. 캐릭터 시스템

### 3.1 캐릭터 구성
- **총 캐릭터 수**: 14명 (확장 예정)
- **등급 체계**: N → R → SR → SSR

### 3.2 등급별 캐릭터

#### SSR 등급 (6명)
| 이름 | 교단 | 속성 | 역할 | 컨셉 |
|-----|------|-----|------|-----|
| 프레야 | 발할라 | Wind | Attacker | "내가 제일 예쁘다구~!" |
| 아마테라스 | 타카마가하라 | Light | Support | 귀찮지만 빛나는 태양신 |
| 제우스 | 올림푸스 | Fire | Attacker | 천둥의 지배자 |
| 토르 | 아스가르드 | Water | Tank | 묠니르의 주인 |
| 이자나미 | 요미 | Dark | Debuffer | 황천의 여신 |
| 오딘 | 아스가르드 | Wind | Support | 지혜의 신 |

#### SR 등급 (5명)
| 이름 | 교단 | 속성 | 역할 |
|-----|------|-----|------|
| 발키리 | 발할라 | Wind | Attacker |
| 스사노오 | 타카마가하라 | Water | Attacker |
| 아테나 | 올림푸스 | Light | Tank |
| 로키 | 아스가르드 | Dark | Debuffer |
| 카구야히메 | 타카마가하라 | Light | Support |

#### R 등급 (3명+)
에인헤리야르, 텐구, 키클롭스, 드워프, 오니, 펜리르

### 3.3 캐릭터 스탯
| 스탯 | 설명 |
|-----|------|
| HP | 체력 |
| ATK | 공격력 |
| DEF | 방어력 |
| SPD | 속도 (턴 순서 결정) |
| CRIT | 치명타 확률 |
| CRIT DMG | 치명타 데미지 |

### 3.4 진화 시스템

| 현재 등급 | 필요 조각 | 필요 골드 | 결과 |
|----------|----------|----------|------|
| N | 10 | 1,000 | R |
| R | 20 | 5,000 | SR |
| SR | 50 | 20,000 | SSR |
| SSR | 100 | 100,000 | SSR+ |

**진화 효과**:
- 기본 스탯 +20%
- 새로운 스킬 해금 (SR 이상)
- 외형 변화

## 4. 전투 시스템

### 4.1 전투 UI 레이아웃
```
┌────────────────────────────────────────────┐
│            턴 순서 바                       │
│  [영웅1] [적1] [영웅2] [적2] [영웅3]        │
├────────────────────────────────────────────┤
│                                            │
│     적 유닛들 (최대 5명)                    │
│     ┌───┐ ┌───┐ ┌───┐                      │
│     │ 👾 │ │ 👾 │ │ 👾 │                      │
│     └───┘ └───┘ └───┘                      │
│                                            │
│     아군 유닛들 (최대 5명)                  │
│     ┌───┐ ┌───┐ ┌───┐                      │
│     │ 🦸 │ │ 🦸 │ │ 🦸 │                      │
│     └───┘ └───┘ └───┘                      │
│                                            │
├────────────────────────────────────────────┤
│  스킬 카드 영역                             │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐       │
│  │스킬│ │스킬│ │ 🔒 │ │ 🔒 │ │ 🔒 │       │
│  │100%│ │ 75%│ │    │ │    │ │    │       │
│  └────┘ └────┘ └────┘ └────┘ └────┘       │
│                                            │
│  [AUTO ON]  [1x] [2x] [3x]     [퇴각]      │
└────────────────────────────────────────────┘
```

### 4.2 턴 순서 시스템
- SPD(속도) 기반 턴 순서 계산
- 상단 바에 행동 순서 표시
- 현재 턴 유닛 하이라이트
- 다음 3턴 미리보기

### 4.3 스킬 카드 시스템
- 하단에 파티 영웅들의 스킬 카드 표시
- 스킬 게이지가 100% 차면 발동 가능
- 터치로 스킬 선택 → 타겟 선택 → 발동
- AUTO 모드: 자동으로 스킬 발동

### 4.4 속성 상성

```
     Fire
    ↗    ↘
Wind  ←  Water

Light ↔ Dark (상호 유리)
```

| 관계 | 데미지 배율 |
|-----|-----------|
| 유리 속성 | 130% |
| 불리 속성 | 70% |
| 동일/무관 | 100% |

## 5. 시너지 시스템

### 5.1 교단 시너지

| 동일 교단 수 | 효과 |
|-------------|------|
| 2명 | ATK +10% |
| 3명 | ATK +15%, DEF +10% |
| 4명 | ATK +20%, DEF +15%, SPD +10% |
| 5명 | 모든 스탯 +25% |

### 5.2 속성 시너지

| 조합 | 효과 |
|------|------|
| Fire x2 | ATK +5% |
| Water x2 | HP +5% |
| Wind x2 | SPD +5% |
| Light x2 | CRIT +5% |
| Dark x2 | CRIT DMG +10% |
| Light + Dark | CRIT DMG +15% |
| 5속성 전부 | 모든 스탯 +15% |

### 5.3 역할 시너지

| 조합 | 구성 | 효과 |
|------|-----|------|
| 균형팀 | 4가지 역할 각 1명 | 모든 스탯 +10% |
| 전멸전략 | Attacker 3명+ | ATK +20%, DEF -10% |
| 철벽수비 | Tank 2명 + Support 2명 | DEF/HP +20% |
| 저주무리 | Debuffer 2명+ | 디버프 효과 +30% |

### 5.4 특수 시너지

| 이름 | 캐릭터 조합 | 효과 |
|------|-----------|------|
| 북유럽의 황혼 | 오딘 + 토르 + 로키 | 전체 스탯 +20% |
| 일본 삼종신기 | 아마테라스 + 스사노오 + 카구야히메 | 스킬 데미지 +25% |
| 올림푸스 최강 | 제우스 + 아테나 | 번개 데미지 +30% |
| 생사의 경계 | 이자나미 + 이자나기 | 부활 확률 20% |

## 6. 장비 시스템

### 6.1 장비 슬롯 (4종)

| 슬롯 | 주요 스탯 | 예시 장비 |
|-----|---------|----------|
| 무기 (Weapon) | ATK, CRIT | 묠니르, 쿠사나기 |
| 방어구 (Armor) | DEF, HP | 황금 양털 |
| 악세서리 (Accessory) | SPD, CRIT DMG | 브리싱가멘 |
| 유물 (Relic) | 특수 효과 | 암브로시아 |

### 6.2 장비 등급
N → R → SR → SSR

### 6.3 대표 장비 목록

| 이름 | 슬롯 | 등급 | 효과 |
|-----|------|-----|------|
| 묠니르 | 무기 | SSR | ATK +150, 번개 데미지 +20% |
| 쿠사나기 | 무기 | SSR | ATK +140, 바람 데미지 +15% |
| 황금 양털 | 방어구 | SSR | DEF +100, HP +500 |
| 브리싱가멘 | 악세서리 | SSR | SPD +30, 매력 +20% |
| 암브로시아 | 유물 | SSR | 턴당 HP 5% 회복 |

### 6.4 장비 강화
- 강화석 + 골드로 레벨업
- 최대 +20 강화

## 7. 스테이지 구성

### 7.1 챕터 구조

| 챕터 | 이름 | 스테이지 수 | 권장 전투력 |
|-----|------|-----------|-----------|
| 1장 | 균열의 시작 | 4 | 100 ~ 1,000 |
| 2장 | 신들의 전장 | 4 | 1,200 ~ 2,500 |
| 3장 | 신화의 종언 | 4 | 3,000 ~ 8,000 |

### 7.2 스테이지 보상

| 스테이지 | 골드 | 경험치 | 첫클리어 보석 |
|---------|-----|-------|-------------|
| 일반 | 100~500 | 50~250 | 30 |
| 보스 | 1,000~10,000 | 500~5,000 | 50~300 |

### 7.3 드롭 아이템
- 경험치 포션 (30~50%)
- 진화 재료 (10~35%)
- 장비 (5~15%)
- 캐릭터 조각 (보스 한정)

## 8. 가챠 시스템

### 8.1 소환 확률

| 등급 | 확률 |
|-----|------|
| SSR | 3% |
| SR | 15% |
| R | 82% |

### 8.2 소환 비용

| 종류 | 비용 |
|-----|------|
| 단차 | 보석 300개 |
| 10연차 | 보석 2,700개 (10% 할인) |
| 티켓 소환 | 소환 티켓 1장 |

### 8.3 천장 시스템
- 90회 소환 시 SSR 확정
- 180회 소환 시 픽업 캐릭터 확정

### 8.4 소환 연출

| 등급 | 연출 |
|-----|------|
| R | 파란색 플래시 |
| SR | 보라색 이펙트 + 진동 |
| SSR | 무지개 이펙트 + 정지 + 스플래시 아트 |
