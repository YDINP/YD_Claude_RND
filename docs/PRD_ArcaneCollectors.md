# PRD: Arcane Collectors (아르케인 컬렉터스)

> 서브컬쳐 스타일의 웹 기반 방치형 수집 RPG

## 1. 개요

### 1.1 프로젝트 정보
- **프로젝트명**: Arcane Collectors (아르케인 컬렉터스)
- **플랫폼**: Web (HTML5 + JavaScript)
- **장르**: 방치형 수집 RPG
- **타겟**: 서브컬쳐/애니메이션 스타일 게임 선호 유저
- **개발 목표**: MVP (2-4주)

### 1.2 컨셉 요약
마법과 던전이 존재하는 판타지 세계에서 다양한 영웅들을 수집하고 육성하여 던전을 탐험하는 방치형 RPG. 블루아카이브, 아크나이츠의 비주얼 스타일과 쿠키런 킹덤, AFK 아레나의 방치형 시스템을 결합한 게임.

### 1.3 레퍼런스
| 게임 | 참고 요소 |
|------|----------|
| 블루아카이브 | UI/UX 디자인, 캐릭터 프레젠테이션 |
| 아크나이츠 | 세계관 깊이, 캐릭터 디자인 |
| AFK 아레나 | 방치형 시스템, 자동 전투 |
| 쿠키런 킹덤 | 수집 시스템, 진화/강화 |

---

## 2. 핵심 게임플레이

### 2.1 방치형 시스템 (Idle System)
```
[오프라인 보상]
- 최대 24시간 오프라인 수익 축적
- 골드, 경험치, 장비 파편 자동 획득
- 접속 시 누적 보상 일괄 수령

[자동 전투]
- 전투 시작 시 AI가 스킬 자동 사용
- 배속 옵션 (1x, 2x, 3x)
- 자동 반복 전투 (스테미나 소진까지)
```

### 2.2 캐릭터 수집 시스템
```
[등급 체계]
- ★★★★★ (5성) - SSR - 1.5%
- ★★★★ (4성) - SR - 8.5%
- ★★★ (3성) - R - 30%
- ★★ (2성) - N - 60%

[소환 시스템]
- 단일 소환: 보석 300개
- 10연차 소환: 보석 2,700개 (10% 할인)
- 천장 시스템: 90회 내 SSR 확정
- 픽업 배너: 특정 캐릭터 확률 UP
```

### 2.3 캐릭터 육성
```
[레벨업]
- 경험치 물약으로 레벨업 (최대 100)
- 레벨업 시 스탯 상승

[스킬 강화]
- 스킬북으로 스킬 레벨업 (최대 10)
- 스킬 효과 및 데미지 증가

[진화/돌파]
- 동일 캐릭터 조각으로 진화
- 최대 6성까지 진화 가능
- 진화 시 스탯 대폭 상승

[장비]
- 무기, 방어구, 악세서리 슬롯
- 장비 강화 (+1 ~ +15)
```

## 3. 게임 콘텐츠

### 3.1 스토리 모드 (메인 던전)
- 챕터별 스테이지 구성 (1-1 ~ 1-10)
- 각 스테이지 3성 클리어 목표
- MVP에서는 1챕터 (10스테이지) 구현

### 3.2 무한 던전 (핵심 방치 콘텐츠)
- 층 수 무제한
- 최고 기록 갱신 시 보상
- 오프라인 시에도 진행

### 3.3 일일 퀘스트
- 매일 갱신되는 임무 5개
- 완료 시 보석, 소환권 보상

## 4. UI/UX 설계

### 4.1 화면 구성
```
┌─────────────────────────────────────────┐
│  [프로필] [설정]      💎 1,500  🪙 50,000│  <- 상단 바
├─────────────────────────────────────────┤
│                                         │
│           [메인 비주얼 영역]              │
│        (캐릭터 일러스트/애니메이션)        │
│                                         │
├─────────────────────────────────────────┤
│  ┌───┐  ┌───┐  ┌───┐  ┌───┐  ┌───┐   │
│  │모험│  │소환│  │영웅│  │상점│  │메뉴│   │  <- 하단 네비게이션
│  └───┘  └───┘  └───┘  └───┘  └───┘   │
└─────────────────────────────────────────┘
```

### 4.2 서브컬쳐 UI 스타일 가이드
```
[컬러 팔레트]
- Primary: #6366F1 (보라/인디고 계열)
- Secondary: #EC4899 (핑크 계열)
- Background: #0F172A ~ #1E293B (다크 테마)
- Accent: #F59E0B (골드/하이라이트)
- Text: #F8FAFC (밝은 텍스트)

[타이포그래피]
- 제목: Noto Sans KR Bold, 24px
- 본문: Noto Sans KR Regular, 14px
- 숫자: 고정폭 폰트 (Roboto Mono)

[디자인 요소]
- 둥근 모서리 (border-radius: 12px)
- 그라데이션 버튼
- 글로우 이펙트 (box-shadow with blur)
- 캐릭터 프레임 애니메이션
- 파티클 이펙트 (소환 연출)
```

### 4.3 주요 화면 상세

#### 메인 로비
- 배경: 판타지 길드 홀 / 기지
- 중앙: 대표 캐릭터 Live2D 스타일 표시
- 터치 시 반응 (인사, 대사 등)

#### 소환 화면
- 화려한 마법진 연출
- 등급별 차별화된 연출 (SSR은 금빛 이펙트)
- 스킵 기능

#### 캐릭터 상세
- 전신 일러스트
- 스탯 그래프 (레이더 차트)
- 스킬 아이콘 및 설명
- 육성 버튼 (레벨업, 스킬강화, 진화)

#### 전투 화면
- 2D 사이드뷰 전투
- HP 바, 스킬 게이지
- 자동 전투 토글
- 배속 버튼

## 5. 기술 스택

### 5.1 프론트엔드
```
- HTML5 + CSS3 + JavaScript (ES6+)
- Phaser 3 (게임 엔진)
- GSAP (애니메이션)
- Howler.js (사운드)
```

### 5.2 데이터 관리
```
- LocalStorage (MVP 단계)
- 향후 확장: Firebase / Supabase
```

### 5.3 에셋
```
- 캐릭터: AI 생성 or 프리 에셋
- UI: 직접 제작 (Figma)
- 이펙트: 파티클 시스템
- BGM/SFX: 프리 사운드
```
