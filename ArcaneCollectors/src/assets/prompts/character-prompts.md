## 스타일 가이드
**트릭컬 리바이브 (Trickle Revive)** 스타일을 기반으로 합니다:
- 귀엽고 발칙한 2D 캐릭터
- 2.5~3등신 SD (Super Deformed) 스타일
- 과장된 표정 (큰 눈, 생동감 있는 표현)
- 밝고 채도 높은 색상
- 동글동글한 외형선
- 장난스럽고 장난꾸러기 같은 분위기

## 프롬프트 사용 가이드

### Midjourney
- `/imagine` 명령어 사용
- 프롬프트를 그대로 복사하여 입력
- `--v 6` 버전 권장
- `--ar 1:1`로 정사각형 비율 유지

### DALL-E 3
- ChatGPT Plus 또는 API 사용
- 자연어 프롬프트를 그대로 입력
- 스타일 일관성을 위해 "Trickle Revive style" 언급

### Stable Diffusion
- Positive와 Negative 프롬프트를 모두 입력
- 권장 설정:
  - Steps: 25-30
  - CFG Scale: 7-9
  - Sampler: DPM++ 2M Karras
  - Model: Anime 계열 체크포인트 (예: AnythingV5, CounterfeitV3)

### 일관성 유지 팁
1. 모든 캐릭터에 동일한 스타일 태그 사용
2. 배경은 흰색 또는 단색으로 유지
3. 2.5 head ratio 비율 강조
