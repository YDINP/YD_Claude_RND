## UI 참조

### 해상도 설정
| 구분 | 기존 | 변경 |
|-----|-----|------|
| 기본 | 480x854 | 720x1280 |
| HD | - | 1080x1920 |
| 스케일링 | 고정 | 반응형 |

### 하단 메뉴 탭
```
┌─────────────────────────────────────┐
│         [게임 콘텐츠 영역]           │
│                                     │
├─────────────────────────────────────┤
│  🏠     ⚔️      📦      🎲      ≡   │
│  홈    모험    가방    소환    더보기 │
└─────────────────────────────────────┘
```

### 메인화면 구성
```
┌─────────────────────────────────────┐
│ 💎 1500  🪙 10000  ⚡ 100/120       │  ← 상단 재화 (에너지 추가)
├─────────────────────────────────────┤
│                                     │
│    [메인 캐릭터 대형 표시]           │
│    (터치 시 대사 + 반응)            │
│                                     │
│  ┌───┐ ┌───┐ ┌───┐ ┌───┐          │  ← 4인 파티
│  │서브│ │서브│ │서브│ │서브│          │
│  └───┘ └───┘ └───┘ └───┘          │
│                                     │
│  [일일퀘스트] [이벤트] [상점]        │  ← 퀵 메뉴
├─────────────────────────────────────┤
│  🏠     ⚔️      📦      🎲      ≡   │
└─────────────────────────────────────┘
```
