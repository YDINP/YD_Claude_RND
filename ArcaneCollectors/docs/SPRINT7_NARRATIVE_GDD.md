### 캐릭터 1 — 아테나 (Athena) / OLYMPUS 교단

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 아테나 (Athena) |
| 교단 | Olympus (황금 교단) |
| 등급 | SSR |
| 직업 | 전략가 / 수호자 |
| 성별 | 여성 |
| 나이 | 불명 (신성 시간 기준 수천 년) |
| 외형 키워드 | 금색 갑옷, 부엉이 문양 방패, 회색 눈동자, 은빛 투구 |

#### 성격 및 배경 스토리

아테나는 올림푸스 교단의 군사 전략을 총괄하는 지혜의 여신이다. 전쟁의 기술과 지식의 신으로서 균열 이전에는 인간의 문명 발전을 조용히 이끌었으나, 균열 이후 급변한 세계 질서에서 올림푸스의 생존을 위한 냉혹한 전략가로 변모했다.

외면은 완벽하다. 단호한 눈빛, 군더더기 없는 판단, 한 치의 오차도 허용하지 않는 전술 수립 능력. 그러나 내면 깊은 곳에서 그녀는 알고 있다 — 어떠한 전략도 '균열'의 근본 원인을 해결하지 못한다는 것을. 지혜의 여신이 해답을 모른다는 것은, 그녀에게 있어 신으로서의 정체성을 무너뜨리는 위기다.

수집가를 처음 만났을 때, 아테나는 경멸했다. 교단에 속하지 않은 자가 세상을 구한다는 발상이 터무니없다고 생각했다. 하지만 수집가가 기존 어느 교단의 논리로도 설명할 수 없는 방식으로 미정령을 연결하는 것을 목격하며, 처음으로 자신의 전략에 없는 변수를 인식하게 된다.

#### 캐릭터 시트

| 항목 | 내용 |
|------|------|
| 동기 (Want) | 올림푸스 교단의 생존과 신성 회복 |
| 욕구 (Need) | 지혜만으로는 해결할 수 없는 문제 앞에서의 겸허함 |
| 두려움 | 무지(無知) — 답을 모르는 상황 |
| 결함 (Flaw) | 완벽주의. 불확실성을 거부하다가 최선의 선택을 놓친다 |

#### 캐릭터 아크

| 구간 | 상태 | 핵심 사건 |
|------|------|---------|
| 1막 (설정) | 냉철한 전략가. 수집가를 도구로 활용하려 한다 | 수집가가 아테나의 전략을 무시하고 더 나은 해법을 찾음 |
| 2막 (발전) | 자신의 전략이 균열에 통하지 않음을 인식, 처음으로 협력을 시도 | 타르타루스의 공격으로 올림푸스 방어선 붕괴 위기 |
| 3막 (해소) | 지혜는 '아는 것'이 아니라 '모르는 것을 인정하는 것'임을 깨닫는다 | 수집가와 함께 올림푸스 최후 방어선 사수 |

#### 대사 톤 가이드

- **어조**: 격식체, 선언적. "~이다", "~해야 한다"
- **특징 표현**: "계산된 결과가 다르다", "이것은 변수가 아니라 오류다", "재고하라"
- **금기어**: 약함, 실수, 두려움 — 절대 자신의 감정을 직접적으로 표현하지 않음

#### 각성 트리거 내러티브 (Awakening Narrative)

**각성 1단계 — "전략의 균열"**
> "내 방패가 이 힘을 받아들이는 날, 나는 처음으로 알게 된다. 지혜란 확신이 아닌 — 의심의 다른 이름이었다는 것을."

수집가가 타르타루스의 봉인을 해제하는 순간, 아테나의 방패에 금이 간다. 완벽한 방어의 상징이 균열을 입는 사건. 이 균열은 그녀에게 외부 위협이 아니라, 스스로의 내면에서 비롯된 것임을 암시한다.

**각성 2단계 — "부엉이의 눈"**
> "나의 부엉이는 밤에만 본다. 나는 낮과 밤 모두를 보아야 한다 — 빛 속의 어둠을, 어둠 속의 빛을."

아테나의 각성은 그녀의 전략 능력이 전투 파티 전체의 시야를 열어주는 능력으로 진화함을 상징한다.

### 캐릭터 2 — 시그룬 (Sigrún) / VALHALLA 교단

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 시그룬 (Sigrún) |
| 교단 | Valhalla (빙강 교단) |
| 등급 | SR |
| 직업 | 발키리 / 전장 안내자 |
| 성별 | 여성 |
| 나이 | 불명 (전사한 영웅 중 최초의 발키리) |
| 외형 키워드 | 백금 날개 갑옷, 붉은 창, 전투 흉터, 불꽃 같은 붉은 머리카락 |

#### 성격 및 배경 스토리

시그룬은 원래 인간이었다. 발할라 교단에서 가장 오래된 발키리 중 하나로, 필사적인 전쟁에서 전사한 후 오딘에게 선택받아 발키리의 날개를 얻었다. 인간이었기에, 그녀는 다른 신들이 이해하지 못하는 것을 안다 — 죽음의 공포와, 그럼에도 싸우는 인간의 의지.

균열 이후, 발할라의 영혼 수확 능력이 감소하자 시그룬은 최전선에 직접 뛰어들었다. 그녀의 방식은 단순하다 — 싸워라, 쓰러져라, 다시 일어나라. 명예는 결과가 아니라 과정에 있다.

수집가를 처음 봤을 때, 시그룬은 한 가지를 확인했다. "죽음이 두렵습니까?" 두려움이 없다고 말하면 거짓말쟁이, 두렵다고 하면 전사의 자격이 있다.

#### 각성 트리거 내러티브

**각성 1단계 — "발키리의 선택"**
> "나는 전장에서 누가 살고 누가 죽는지를 골랐다. 이제 나는 선택한다 — 모두가 살아남는 전장을."

시그룬의 각성은 '선택적 수호'에서 '전체 수호'로의 진화를 상징한다.

### 캐릭터 3 — 모리간 (Morrígan) / AVALON 교단

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 모리간 (Morrígan) |
| 교단 | Avalon (보랏빛 교단) |
| 등급 | SSR |
| 직업 | 예언자 / 마법 직조자 |
| 성별 | 여성 |
| 나이 | 불명 (아서왕 시대부터 존재) |
| 외형 키워드 | 까마귀 날개 망토, 보라색 룬 문신, 반쪽 검은 눈, 반쪽 보라색 눈 |

#### 성격 및 배경 스토리

모리간은 켈트 신화의 전쟁과 운명의 여신이다. 아발론 교단의 핵심 목표는 '새로운 신성의 창조' — 기존 신들이 죽어가는 균열의 시대에, 아무것도 없는 곳에서 새로운 신성의 불꽃을 피우는 것이다. 모리간은 이 프로젝트의 설계자이자 관찰자다.

그녀는 혼돈 속에서 패턴을 본다. 까마귀 형태로 전장을 날아다니며 죽음과 탄생의 경계를 관찰한다. 감정을 드러내지 않지만, 그것은 무관심이 아니라 — 모든 것에 동등하게 관심을 두기 때문이다. 수집가를 보았을 때, 모리간은 예언의 흐름에서 벗어난 존재를 처음으로 발견했다. 예측 불가능한 자. 이것은 위협인가, 아니면 기회인가.

#### 각성 트리거 내러티브

**각성 1단계 — "운명의 재직조"**
> "나는 천 년을 예언했다. 이제야 나는 예언하지 않는 자 하나를 만났다. 그것이 나를 처음으로 놀라게 했다."

### 캐릭터 4 — 아마테라스 (Amaterasu) / TAKAMAGAHARA 교단

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 아마테라스 (Amaterasu) |
| 교단 | Takamagahara (백금 교단) |
| 등급 | SSR |
| 직업 | 광명 여신 / 정화자 |
| 성별 | 여성 |
| 나이 | 불명 (태양과 함께 태어난 존재) |
| 외형 키워드 | 백색 기모노, 금빛 태양 오비, 눈을 감은 채로 보는 자 |

#### 성격 및 배경 스토리

아마테라스는 다카마가하라의 최고신이다. 빛과 정화의 여신으로, 균열로 인해 오염된 신성 에너지를 정화하는 능력이 세상에서 가장 필요한 능력이 되었다. 그러나 그 능력에는 대가가 있다 — 오염을 정화할수록 그녀 자신도 균열의 영향을 받기 시작한다.

아마테라스는 희생을 담담하게 받아들인다. 그것이 신의 소명이라 믿기 때문이다. 하지만 수집가를 만나면서, 그녀는 처음으로 질문을 던진다 — "희생 없이도 세상을 구할 수 있다면, 그것이 더 나쁜 일인가?"

#### 각성 트리거 내러티브

**각성 1단계 — "빛의 반면"**
> "빛이 강할수록 그림자도 짙어진다. 나는 빛을 포기하는 대신 — 그림자를 내 것으로 만들기로 했다."

### 캐릭터 5 — 흑제 (黑帝, Hēi Dì) / KUNLUN 교단

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 흑제 (黑帝) / 별칭: 현무(玄武)의 목소리 |
| 교단 | Kunlun (비취 교단) |
| 등급 | SR |
| 직업 | 도사 / 자연 조화자 |
| 성별 | 남성 |
| 나이 | 불명 (수천 년의 수행) |
| 외형 키워드 | 흑색 도포, 거북이 문양 지팡이, 짧은 흑발, 항상 웃는 눈 |

#### 성격 및 배경 스토리

흑제는 곤륜산(昆侖山) 선계의 도사로, 도(道)의 자연스러운 흐름을 지키는 역할을 맡고 있다. 균열이 발생했을 때, 흑제의 반응은 다른 교단과 달랐다 — 아무것도 하지 않았다. 도(道)는 억지로 흐르게 할 수 없다. 적절한 때를 기다리는 것, 그것도 하나의 방법이다.

그러나 균열이 자연의 흐름 자체를 뒤틀기 시작하면서, 흑제는 처음으로 개입을 결정했다. 그의 방식은 여전히 느리고 조용하지만, 그가 개입할 때 세상이 달라진다. 수집가를 보며 흑제는 말했다: "자네는 강물을 막으려 하는가, 아니면 강물의 방향을 바꾸려 하는가?"

#### 각성 트리거 내러티브

**각성 1단계 — "무위의 선택"**
> "나는 아무것도 하지 않는 것을 선택했다. 그것이 최선의 행동이었다. 그런데 이제, 나는 행동하기로 선택한다. 이것 역시 — 도(道)다."

## 섹션 2. 프롤로그 3챕터 스크립트

### 세계관 진입 설계 원칙

- **플레이어 정체성 확립**: 수집가(Collector)가 누구인지, 왜 특별한지를 자연스럽게 설명
- **교단 충돌 서사**: 처음부터 교단들이 왜 갈등하는지 보여줌
- **수집 동기 부여**: 미정령을 모아야 하는 감정적 이유 제공
- **연출 방식**: 텍스트 대화창 + 캐릭터 실루엣 (비주얼 노벨 형식)

---

### 프롤로그 챕터 1 — "균열의 목격자"

**배경**: 균열 직후의 황폐한 세계. 하늘이 보라색으로 갈라진 풍경.
**감정 목표**: 위기감 + 호기심 (세계가 무너지고 있다는 실감)
**등장 인물**: 내레이터, 정체불명의 목소리

---

**[화면: 검은 화면에서 서서히 갈라진 하늘로 페이드인]**

*"이 이야기는 끝에서 시작한다."*

*"균열(The Fracture)이 세계를 가른 이후, 신성의 에너지는 조금씩, 조금씩 사라졌다."*

*"신들은 두려워했다. 인간들은 모른 척했다. 그리고 미정령들은 — 그저 방황했다."*

**[화면: 빛나는 작은 존재들이 이리저리 떠다니는 모습 — 미정령들]**

"거기 — 당신."

"당신은 보이는군요. 우리가."

**[플레이어 선택지 없음 — 서사 진행]**

**[내레이터]**
*"그것이 시작이었다. 보통의 인간에게는 보이지 않는 미정령의 모습이, 당신에게는 선명하게 보였다."*

*"왜인지는 아무도 모른다. 그것이 운명인지, 저주인지, 아니면 단순한 우연인지."*

*"하지만 분명한 것은 — 당신은 이 세계에서 유일하게, 그들의 목소리를 들을 수 있다."*

**[화면: 미정령 하나가 플레이어 앞에 멈추는 모습]**

**[미정령]**
*"데려가줘. 어디로든. 나는 혼자 있는 게 무서워."*

**[튜토리얼 연계: 첫 번째 미정령 수집 — 가챠 연출로 전환]**

### 프롤로그 챕터 2 — "교단의 첫 번째 목소리"

**배경**: 올림푸스와 발할라의 경계 지역, 신성 에너지 충돌 구역
**감정 목표**: 갈등 이해 + 수집가로서의 고유한 위치 인식
**등장 인물**: 아테나, 시그룬, 수집가(플레이어)

---

**[화면: 두 신이 서로를 마주보고 있는 장면]**

**[아테나]**
