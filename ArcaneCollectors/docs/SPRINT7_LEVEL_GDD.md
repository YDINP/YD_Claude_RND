## 레벨 디자인 원칙

ArcaneCollectors는 자동전투 수집형 RPG이므로, 공간적 맵 레이아웃보다 **전투 파라미터 설계와 진행 흐름**이 레벨 디자인의 핵심이다.

### 핵심 레벨 설계 축

1. **교단 보너스 연계**: 스테이지별 유리 교단 지정 → 수집 동기 강화
2. **난이도 곡선**: 3구간 원칙 (도입부 10~15% / 발전부 60~70% / 절정부 15~25%)
3. **보상 주기**: 5스테이지마다 강화된 보상 → 진행 동기 유지
4. **무한 콘텐츠**: Abyss Tower로 엔드게임 처리

### 1-2. 난이도 곡선 설계

#### 3구간 원칙 적용

**도입부 (챕터 1~2, 전체 스테이지의 22%):**
- 목적: 기본 전투 루프 + 교단 시너지 학습
- 난이도 지표: 파티 평균 ATK 기준 적 체력 배율 1.5x~2.5x
- 새 메커니즘 도입: 챕터 1은 일반 전투만, 챕터 2에서 엘리트 전투 도입

**발전부 (챕터 3~7, 전체 스테이지의 56%):**
- 목적: 스킬 조합 + 교단 보너스 활용 필수화
- 난이도 패턴: 홀수 챕터 = 상승 구간, 짝수 챕터 = 소폭 완화 후 재상승
- 신규 요소: 챕터 3 마법 버프 적, 챕터 5 자연 재생 적, 챕터 7 동결 디버프 적

**절정부 (챕터 8~9, 전체 스테이지의 22%):**
- 목적: 모든 교단 시너지 통합 운용 필수
- 난이도 지표: 파티 평균 ATK 기준 적 체력 배율 8x~12x
- 특수 메커니즘: 챕터 8 균열 폭발 피해(전파티), 챕터 9 페이즈 전환 보스

### 1-4. 교단 보너스 연계 설계

**스테이지 유리 교단 설정 원칙:**
- 챕터당 1개 교단이 +25% ATK/DEF 보너스
- 중립 교단(Balance, Chaos, Nature)은 해당 교단이 없으면 전체 +10% 보너스
- 챕터 8~9: 복수 교단 조합 보너스 (2교단 이상 혼합 파티 시 +15%)

**보너스 체감 설계:**
- 보너스 없이도 클리어 가능하되, 보너스 있으면 체감 난이도 1~2단계 감소
- 플레이어가 "저 챕터는 XX 교단 캐릭터를 키워야겠다"는 목표 동기 부여

## 섹션 2. 무한 타워 (Abyss Tower) 구조 설계

### 2-1. Abyss Tower 개요

| 항목 | 내용 |
|------|------|
| 해금 조건 | 챕터 3-3 클리어 |
| 최대 층 | 무제한 (자동 생성) |
| 초기화 주기 | 매주 월요일 오전 5:00 (개인 기록 초기화) |
| 진행 방식 | 층 순차 도전, 중간 저장 없음 (한 번에 연속 진행) |
| 재도전 조건 | 실패 시 즉시 재도전 가능, 해당 층부터 재시작 |
| 교단 보너스 | 없음 (순수 파티 전투력 + 스킬 운용 평가) |

### 2-2. 층 구성 패턴

**기본 층 패턴 (5층 단위 사이클):**

| 층 유형 | 출현 빈도 | 구성 | 특징 |
|--------|---------|------|------|
| 일반 전투 | 3층/5층 | 적 3~5마리 | 표준 전투 |
| 엘리트 전투 | 1층/5층 | 강화 적 2~3마리 | 특수 패시브 보유 |
| 보스 전투 | 1층/5층 (5의 배수 층) | 보스 1마리 + 소환물 | 특수 메커니즘 |

**10층 단위 난이도 점프:**

| 층 구간 | 적 전투력 배율 | 특수 메커니즘 |
|--------|------------|-----------|
| 1~10층 | 기준 1.0x | 없음 |
| 11~20층 | 1.5x | 적 패시브 스킬 활성 |
| 21~30층 | 2.2x | 균열 피해 추가 (-3%HP/턴) |
| 31~50층 | 3.0x | 다중 보스 (2마리 동시) |
| 51~70층 | 4.5x | 파티 버프 제한 (버프 지속 1턴 감소) |
| 71~100층 | 7.0x | 페이즈 전환 보스, 분열 기믹 |
| 100층+ | 10x+ | 무작위 혼합 기믹 |

### 2-3. 특수 층 설계

**25층, 50층, 75층, 100층 — "균열 심층부 보스":**
- 해당 챕터 보스의 강화판 등장
- 클리어 시 특별 보상 (SSR 각성 소재 확정)

**기믹 층 유형 (50층부터 간헐적 등장):**
| 기믹 | 설명 | 대응 전략 요구 |
|------|------|------------|
| 분열형 | 처치 시 2마리로 분열 | 교단 시너지 광역 스킬 |
| 회생형 | HP 0 시 1회 HP 30% 회복 | 딜 집중 후 확인 사살 |
| 반사형 | 받은 피해의 20% 반사 | 방어 집중 파티 구성 |
| 흡혈형 | 적이 준 피해의 30% 흡수 | 버프 스킬 최소화 |

### 2-4. 주간 초기화 및 보상 체계

**주간 도달 층 기준 보상:**

| 도달 층 | 보상 |
|--------|------|
| 10층 | 에너지 보석 ×15, 골드 ×3,000 |
| 20층 | 에너지 보석 ×25, 각성 소재(하) ×2 |
| 30층 | 고급 가챠 재화 ×1, 각성 소재(중) ×2 |
| 50층 | 고급 가챠 재화 ×3, SSR 각성 소재 ×1 |
| 75층 | 고급 가챠 재화 ×5, SSR 각성 소재 ×2 |
| 100층 | 시즌 한정 타이틀 + 고급 가챠 재화 ×10 |

**도달 기록 유지**: 주간 초기화되더라도 **개인 최고 기록**은 영구 보존 (프로필 표시용)

## 섹션 4. 이벤트 던전 확장 계획 (3종 → 6종)

### 4-2. 신규 이벤트 던전 3종 추가

| 항목 | 내용 |

## 섹션 5. 레벨 플로우 검증 체크리스트

### 5-1. 챕터 진행 흐름

```
□ 챕터 1~2에서 신규 플레이어가 30분 이내 첫 클리어 가능한가?
□ 연속 3개 이상 클리어 불가 스테이지 배치 없음 (스파이크 방지)
□ 각 챕터 보스 전 스테이지에서 파티 전투력 목표치 안내가 있는가?
□ 교단 보너스를 모르는 플레이어도 챕터 6까지 클리어 가능한가?
□ 챕터 9 최종 보스가 메인 스토리의 감정적 클라이맥스를 제공하는가?
```

### 5-2. Abyss Tower

```
□ 100층 돌파가 최소 3개월 이상 플레이 목표로 설정되었는가?
□ 30~50층 구간이 캐주얼과 하드코어의 분기점이 되는가?
□ 주간 초기화 전 '오늘 몇 층까지 갔나' 확인 UX가 명확한가?
□ 기믹 층이 예고 없이 등장해 불공정한 실패감을 주지 않는가?
```

```
□ 길드 비가입 플레이어도 의미 있는 솔로 콘텐츠가 충분한가?

□ 길드 클리어 보너스가 기여도 낮은 길드원도 참여를 유지하게 하는가?
```
