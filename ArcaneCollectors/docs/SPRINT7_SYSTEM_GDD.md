## 게임 루프 재설계

### Core Loop (핵심 루프, 2~5분 사이클)

```
에너지 소비 → 스테이지 전투 (스킬 발동 포함) → 경험치/재화 획득
→ 캐릭터 성장(레벨업/각성) → 더 강한 스테이지 도전 → (반복)
```

### Meta Loop (메타 루프, 일/주 단위 사이클)

```
일일 퀘스트 완료 → 일일 재화 수급 → 주간 랭킹 보상 → 가챠 재화 누적
→ 신규 캐릭터 획득 → 교단 시너지 완성 → 신규 챕터 해금 → (반복)
```

### Engagement Loop (재방문 유인)

```
출석 체크 → 일일 퀘스트 알림 → 에너지 완충 알림 → 이벤트 기간 한정 보상
→ 배틀패스 진행률 → 길드 레이드 참여 → (반복)
```

**스킬 유형 분류:**

| 유형 | 발동 조건 | 대상 | 비고 |
|------|---------|------|------|
| 액티브 스킬 | 쿨타임 기반 (N턴마다 자동 발동) | 단일/전체/교단 | 전투 중 자동 발동 |
| 패시브 스킬 | 항상 적용 | 자신 | 스탯 보정 상시 적용 |
| 반응 스킬 | 특정 조건 달성 시 (HP 50% 이하 등) | 단일/전체 | 조건부 발동 |
| 교단 시너지 스킬 | 같은 교단 N명 파티 시 해금 | 파티 전원 | 파티 구성 보너스 |

### 1-2. 스킬 발동 우선순위 및 턴 처리

```
턴 시작
  ↓
1. 패시브 스킬 효과 적용 (스탯 계산)
  ↓
2. 반응 스킬 조건 체크 (HP 임계값 등)
  ↓
3. 액티브 스킬 쿨타임 감소 및 발동 여부 판정
   - 쿨타임 = 0 이면 즉시 발동
   - 복수 캐릭터 동시 발동 시 ATK 순으로 처리
  ↓
4. 기본 공격 (스킬 미발동 캐릭터)
  ↓
5. 교단 시너지 체크 및 보너스 적용
  ↓
6. 적 행동 처리
  ↓
7. 버프/디버프 지속시간 감소
  ↓
다음 턴
```

### 1-3. 피해량 계산 공식

```
기본 피해량 = ATK × 스킬배율 × (1 - DEF / (DEF + 방어계수))
  - 방어계수 = 500 (고정값, 스테이지 스케일링 없음)
  - 스킬배율: 액티브 스킬 1.5x~4.0x / 기본 공격 1.0x

크리티컬 피해량 = 기본 피해량 × (1 + CRIT_DMG)
  - 기본 CRIT_DMG = 0.5 (50% 추가)
  - CRIT_RATE 기본값 = 5%

최소 피해량 = max(1, 기본 피해량)

교단 보너스 피해량 = 기본 피해량 × (1 + 교단 스테이지 보너스%)
  - 스테이지별 유리 교단: +25% 추가 피해
```

### 1-4. 스킬 등급별 배율 기준

| 캐릭터 등급 | 액티브 스킬 배율 | 쿨타임 | 패시브 보정 |
|------------|--------------|--------|----------|
| N (일반) | 1.5x | 4턴 | 스탯 +3% |
| R (레어) | 2.0x | 4턴 | 스탯 +5% |
| SR (슈퍼레어) | 2.8x | 3턴 | 스탯 +8% |
| SSR (슈퍼슈퍼레어) | 4.0x | 3턴 | 스탯 +12% |

### 1-5. 교단 시너지 스킬 조건

| 파티 내 교단 인원 | 시너지 효과 |
|----------------|----------|
| 동일 교단 2명 | ATK +10% |
| 동일 교단 3명 | ATK +10%, DEF +10% |
| 동일 교단 4명 (풀 파티) | ATK +15%, DEF +15%, 교단 전용 스킬 해금 |

- 주간 퀘스트 완료: 완료당 200 EXP
- 이벤트 참여: 이벤트별 별도 책정

**목표**: 일일 퀘스트 전부 완료 플레이어는 8주 시즌 내 60티어 완주 가능
- 일일 퀘스트 5개 × 50 EXP × 56일 = 14,000 EXP
- 스테이지 일일 3회 × 10 EXP × 56일 = 1,680 EXP
- 주간 퀘스트 7개 × 200 EXP × 8주 = 11,200 EXP
- **총 약 26,880 EXP** (60티어 완주 기준 24,000 EXP)
- 여유율 약 12% 확보 → 결석 허용 버퍼 존재

## 섹션 5. 밸런스 검증 체크리스트

### 5-1. 스킬 밸런스

```
□ SSR 캐릭터가 SR의 2배 이상 효율을 내지 않는다 (약 1.4x 목표)
□ 파티 4인 모두 SSR 없이도 챕터 6까지 클리어 가능
□ 교단 시너지가 있을 때와 없을 때 효율 차이: 최대 30% 이내
□ 패시브 스킬이 모든 전투 상황에서 동등하게 유효한가 (특정 상황 독점 금지)
```

### 5-2. 경제 밸런스

```
□ 무과금 플레이어가 주요 스토리 챕터를 에너지 부족 없이 즐길 수 있는가
□ 골드 인플레이션: 레벨 50 캐릭터 10명 보유 시 골드가 과잉되지 않는가
□ 각성 소재 수급량이 과금 없이 주당 1회 각성 가능한 수준인가
□ 배틀패스 가격이 SSR 기대 과금액의 75% 이하인가 (비용 효율 우위 확보)
```

### 5-3. 일일 루틴 부하

```
□ 일일 퀘스트 5개 완료 예상 시간: 15~20분 이내
□ 주간 퀘스트 7개 완료 예상 추가 시간: 30분 이내 (주간 기준)
□ 출석 체크 포함 최소 접속 의미 있는 보상 확보 (1분 접속도 의미 있어야 함)
□ 퀘스트 스킵/자동 완료 기능 없음 (직접 플레이 유도 원칙)
```
