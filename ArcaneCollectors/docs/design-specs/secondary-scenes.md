# 보조 씬 디자인 스펙

> **대상**: QuestScene, TowerScene, InventoryScene, SettingsScene
> **기준**: `DESIGN_SYSTEM.md` (v1.0)
> **해상도**: 1080x1920 (9:16), `s()` / `sf()` 스케일 함수 사용

---

## 공통 규격

### TopBar (모든 씬 동일)

| 속성 | 스펙 | 현재 코드 | 수정 필요 |
|------|------|----------|----------|
| 높이 | `s(80)` (DESIGN_SYSTEM 기준) | `s(100)` | Yes - `s(80)`으로 통일 |
| 배경 | `0x0F172A` alpha 0.95 | 일치 | No |
| 하단 구분선 | `COLORS.primary` alpha 0.3 | alpha 0.5 | Yes - 0.3으로 통일 |
| 제목 폰트 | `sf(24)` bold, `#F8FAFC` | 일치 | No |
| 뒤로가기 | `sf(14)`, `#94A3B8`, `← 뒤로` | InventoryScene은 Arial 사용 | Yes - Noto Sans KR 통일 |
| 뒤로가기 터치영역 | 최소 `s(44)` x `s(44)` | `s(50)` x `s(40)` | Yes - `s(50)` x `s(44)` |

### 콘텐츠 시작 Y

TopBar `s(80)` + 간격 `s(20)` = **`s(100)`** (기존 `s(120)` → 조정)

### 폰트 패밀리 통일

모든 텍스트: `'"Noto Sans KR", sans-serif'` (InventoryScene의 `'Arial'` 수정 필요)

## 수정 필요 사항 요약

### 공통
1. **TopBar 높이**: `s(100)` → `s(80)` (DESIGN_SYSTEM 기준)
2. **TopBar 구분선 alpha**: 0.5 → 0.3
3. **뒤로가기 터치영역**: height `s(40)` → `s(44)` (최소 터치 규격)
4. **콘텐츠 시작 Y**: TopBar 변경에 맞춰 조정

### QuestScene
5. **진행바 색상**: 이미 `COLORS.success` / `COLORS.primary` 사용 - 일관성 확인 완료

### TowerScene
6. **층수 카드 배경**: 단색 원 → 그라데이션 효과 추가로 깊이감 부여
7. **진행바 배경**: `0x1E293B` 사용 (DESIGN_SYSTEM의 `0x374151` 권장과 불일치)

### InventoryScene
8. **폰트 통일**: 모든 `'Arial'` → `'"Noto Sans KR", sans-serif'`
9. **등급 프레임**: 현재 단색 → DESIGN_SYSTEM의 border/background/glow 분리 적용
10. **TopBar 색상 표현**: `COLORS.text.toString(16)` 동적 변환 → CSS hex 직접 사용

### SettingsScene
11. **토글 스위치**: 이미 디자인 시스템 규격과 거의 일치 - 미세 조정만 필요
