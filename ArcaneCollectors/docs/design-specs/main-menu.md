## 3. 개선 사항 요약

### 3.1 색상 통일
- TopBar 배경: `COLORS.backgroundLight` → `COLORS.bgDark` (DESIGN_SYSTEM 4.1 기준)
- TopBar depth: `Z_INDEX.UI` → `Z_INDEX.TOP_BAR` (300)
- TopBar 하단 구분선 추가 (primary alpha 0.3)
- 하드코딩 hex 문자열 → `COLORS.*` 상수 사용

### 3.2 간격 일관성
- 모든 패널 좌우 여백: s(20) 통일
- 패널 간 간격: s(0)~s(5) 범위로 밀착 (현재 이미 양호)

### 3.3 폰트 규격
- fontFamily: 'Arial' → '"Noto Sans KR", Arial' 통일 (DESIGN_SYSTEM 2.1)

### 3.4 인터랙션 피드백
- 하단 메뉴: hover 시 원형 크기 변경 → scale tween으로 변경
- 하단 메뉴: pointerdown 시 scale 0.92 추가
- 설정 버튼: hover 피드백 개선

### 3.5 TopBar 정렬
- 구분선 추가로 콘텐츠 영역과 명확히 분리
- depth를 TOP_BAR(300)으로 상향
