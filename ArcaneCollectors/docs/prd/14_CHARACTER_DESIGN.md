> 버전: v1.0

## 14.1 디자인 원칙

### 아트 스타일
| 항목 | 규격 |
|------|------|
| **등신 비율** | 2.5~3등신 (치비/SD 스타일) |
| **기본 체형** | 여성·소년: 2.5등신 / 남성·거구: 3등신 |
| **외곽선** | 두꺼운 아웃라인 (2~3px), 밝은 색상 강조 |
| **표정** | 과장된 이모 표현 (큰 눈, 풍부한 표정) |
| **질감** | 셀 셰이딩 + 소프트 그라데이션 하이브리드 |

### 디자인 3원칙
1. **실루엣 구별**: 같은 교단 내에서도 실루엣만으로 캐릭터를 구분 가능해야 함
2. **분위기 표현**: 분위기(Mood) 컬러가 의상·이펙트에 자연스럽게 녹아들어야 함
3. **문화 존중**: 각 교단의 신화적 배경을 왜곡 없이 반영

| **★3 (R)** | ★★★ | 파랑 테두리 + 미세 장식 | 별 무늬 배경 | 파란 빛줄기 |
| **★4 (SR)** | ★★★★ | 보라 테두리 + 교단 문양 | 교단 문양 배경 | 보라 마법진 + 교단 심볼 |

### ★5 소환 연출 상세
1. **암전** (0.5초)
2. **분위기 이펙트** — 해당 분위기의 파티클이 화면 중앙으로 수렴 (1초)
3. **교단 심볼** — 배경에 교단 문양이 떠오름 (0.5초)
4. **캐릭터 등장** — 카드가 뒤집히며 캐릭터 실루엣→풀컬러 (1초)
5. **명대사** — `voiceLines.summon` 텍스트 표시 (1.5초)
6. **페이드아웃** — 결과 화면으로 전환 (0.5초)

## 14.6 디자인 데이터 스키마

### characters.json `design` 필드 표준

```json
{
  "design": {
    "style": "string",           // 캐릭터 고유 디자인 테마 (예: "trickle_revive", "ancient_titan")
    "bodyRatio": "string",       // 등신 비율 (예: "2.5 head", "3 head")
    "hairColor": "#hex",         // 머리카락 색상
    "hairStyle": "string",       // 머리 스타일 설명 (한국어)
    "eyeColor": "#hex",          // 눈 색상
    "eyeStyle": "string",        // 눈 스타일 설명 (한국어)
    "primaryColor": "#hex",      // 주요 색상 (의상/갑옷)
    "secondaryColor": "#hex",    // 보조 색상
    "accentColor": "#hex",       // 강조 색상 (장식/이펙트)
    "outfit": "string",          // 의상 설명 (한국어)
    "accessories": ["string"],   // 액세서리 목록 (2~3개, 한국어)
    "signaturePose": "string",   // 대표 포즈 설명 (한국어)
    "chibiFeatures": "string",   // 치비 강조 요소 (한국어)
    "height": "string",          // 키 (예: "175cm")
    "age": "string"              // 나이 (예: "외형 20세", "수천년" 등)
  }
}
```

### 필드 가이드라인
- `style`: 캐릭터의 전체적 디자인 방향을 한 단어로. 교단+분위기 조합 반영
- `primaryColor`: 교단 테마 컬러를 기본으로, 분위기 컬러와 조화
- `accentColor`: 분위기(Mood) 컬러에서 가져오기 권장
- `accessories`: 무기는 skills에 포함되므로, 장신구/특수 아이템 위주
- `chibiFeatures`: SD/치비화 시 강조할 포인트 (과장 가능)

| Calm | 초록 파동 | 수면 결계 | 치유의 비 | 잔잔한 호수 + 달빛 |
| Stoic | 회색 충격 | 대지 균열 | 강철 방벽 | 산맥이 솟아오르는 배경 |
| Devoted | 핑크 빛줄기 | 치유 서클 | 축복의 날개 | 꽃잎이 날리는 신전 |
| Cunning | 파랑 레이저 | 빙결 트랩 | 정밀 관통 | 서리 낀 전장 |
| Noble | 금색 검기 | 왕의 칙령 (범위) | 성스러운 심판 | 황금 왕좌 + 오라 |
| Mystic | 보라 마법진 | 별빛 폭발 | 차원 균열 | 우주/성좌 배경 |

### 14.7.2 시너지 스킬 연출

| 시너지 종류 | 발동 연출 |
|------------|----------|
| **교단 시너지** (동일 교단 2~4명) | 교단 심볼이 파티 위에 떠오르며 오라 버프 |
| **분위기 시너지** (조합 발동) | 해당 분위기 컬러들이 합쳐지며 새로운 이펙트 |
