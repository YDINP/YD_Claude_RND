## 1. 시스템 개요

### 핵심 컨셉

> **"같은 사람이 다른 길을 걷는다. 그 길이 운명을 결정한다."**

기존 v1.0 시스템은 교단+클래스 조합이 랜덤으로 영웅을 결정했다. 플레이어는 어떤 캐릭터가 나올지 모르는 상태에서 가챠를 돌렸으며, 캐릭터 자체에 대한 정서적 애착을 형성하기 어려웠다.

v2.0에서는 **기본영웅이 고유한 이름과 정체성을 가진 캐릭터**로 먼저 존재하고, 플레이어가 **어떤 기관에 입문시키느냐에 따라 같은 캐릭터가 다른 형태로 각성**한다. 이 구조는 다음 세 가지를 달성한다:

| 목표 | 달성 방법 |
|------|----------|
| **캐릭터 애착** | 기본영웅 단계에서 이름/배경/성격 제공 → 플레이어가 "이 캐릭터를 어떻게 키울까" 고민 |
| **전략적 선택** | 기관 선택 = 스킬 방향성과 역할 결정 → 같은 캐릭터의 다른 빌드로 파티 구성 전략 다양화 |
| **장기 수집 동기** | 한 기본영웅으로 여러 기관 루트 개방 가능 → 컬렉션 완성 욕구 자극 |

### v1.0과의 주요 차이

| 항목 | v1.0 | v2.0 |
|------|------|------|
| 캐릭터 정체성 | 교단+클래스 조합으로 결정 | 기본영웅이 먼저 존재 |
| 전직 결과 | 가챠로 랜덤 결정 | 기관 선택으로 확정 |
| 수집 대상 | 전직 영웅 카드 | 기본영웅 + 기관 진화루트 |
| 이름 체계 | "Zeus (제우스)" 독립 존재 | "번개의 아이리스" = 아이리스 + 올림푸스 |
| 플레이어 감정 | "좋은 SSR 뽑았다" | "내 아이리스를 이렇게 키웠다" |

## 2. 캐릭터 계층 구조

### 2-1. 전체 계층도

```
[미정령 — Unbound Spirit]
  형태 불명, 어느 기관에도 속하지 않은 원초적 에너지체
  전투 투입 가능 (기본 능력치)
  최대 레벨 15, 능력치 35% 발현
  → 조각 30개 + 정령석 ×3 → 기본영웅으로 각성

[기본영웅 — Base Hero]
  고유 이름/정체성을 가진 캐릭터
  이름: 아이리스, 루카, 카이, 세라, 린, 오마르, 솔, 하나, 레온, 파올로 (등)
  최대 레벨 30, 능력치 55% 발현
  전투 투입 가능 (추천 라인업)
  → 기관 각인서 ×1 + 기관별 에센스 ×20 → 전직 영웅으로 진화

[전직 영웅 — Ascended Hero]
  기본영웅 + 기관의 결합체
  이름 규칙: "[수식어]의 [기본영웅이름]" 또는 "[기관형용사] [기본영웅이름]"
  예: 번개의 아이리스 / 성약의 아이리스 / 암흑의 아이리스
  SSR: 최대 레벨 60, 100% 능력치 + 전용 각성 스킬
  SR:  최대 레벨 50, 82% 능력치
  R:   최대 레벨 40, 65% 능력치
```

### 2-2. 미정령 획득 및 각성

#### 미정령 획득 경로

| 획득 경로 | 수량 | 비고 |
|----------|-----|------|
| 스테이지 클리어 보상 | 1~3개 | 일반 전투 |
| 일일 퀘스트 완료 | 2~4개 | 매일 초기화 |
| 이벤트 교환소 | 한정 수량 | 이벤트 기간 |
| 가챠 (특별 배너) | 확정 1개 | 10연 기준 |
| 탑/레이드 보상 | 3~10개 | 고난이도 |

미정령은 **특정 기본영웅과 연결되지 않은** 범용 재화다. 기본영웅 각성 시 소모된다.

#### 기본영웅 각성 조건

```
미정령 조각 30개 (기본영웅별 독립 카운트)
+ 정령석 ×3

→ 해당 기본영웅 1명 획득
   (미정령 조각은 특정 기본영웅에 귀속됨)
```

**미정령 조각 획득 방법**:
- 배너에서 해당 기본영웅 "조각" 형태로 등장 (일반 가챠)
- 교환소에서 범용 조각권으로 교환
- 특정 스테이지 클리어 보상 (고정 드롭)

### 2-3. 기관 진화 메커니즘

#### 진화 루트 수

기본영웅마다 **2~3개의 진화 루트**를 보유한다.

| 기본영웅 등급 | 진화 루트 수 | 설계 의도 |
|------------|------------|---------|
| 일반 기본영웅 | 2개 | 명확한 A/B 선택, 집중 투자 |
| 희귀 기본영웅 | 3개 | 폭넓은 파티 활용, 수집 완성 욕구 |
| (미래 확장) 전설 기본영웅 | 최대 4개 | 특별 이벤트 한정 |

**이유**: 기관이 12개이지만 기본영웅마다 모든 기관과 호환되지 않는다. 각 기본영웅의 성격/배경이 특정 기관 2~3개와 자연스럽게 연결된다. 나머지 기관과의 진화는 "설정상 불가능한 조합"으로 처리하여 세계관 일관성을 유지한다.

#### 기관 호환성 원칙

```
기본영웅 '아이리스' (번개 친화적, 의지 강함)
  호환 기관: olympus (번개), valhalla (전사), chaos (역설적 전환)
  비호환: nature (부드러운 자연과 충돌), avalon (성스러운 서약과 불화)

기본영웅 '세라' (치유 본능, 공감 능력)
  호환 기관: avalon (성약), kunlun (선계 의술), nature (생명의 순환)
  비호환: yomi (죽음과 정반대), chaos (예측 불가능성이 치유와 충돌)
```

#### 진화 재화 상세

| 재화 | 설명 | 획득 경로 |
|------|------|---------|
| **기관 각인서** | 기관 진화 시작권 (기관별 공통) | 가챠, 탑 클리어, 월간 패스 |
| **기관 에센스** | 기관별 전용 재화 (12종) | 해당 기관 관련 스테이지, 이벤트 |
| **각성의 불꽃** | SSR 등급 진화 시 추가 필요 | 레이드, 프리미엄 교환소, 결제 |

#### 등급별 진화 비용

| 결과 등급 | 기관 각인서 | 기관 에센스 | 각성의 불꽃 |
|---------|-----------|-----------|-----------|
| R 전직 영웅 | ×1 | ×10 | 불필요 |
| SR 전직 영웅 | ×1 | ×20 | 불필요 |
| SSR 전직 영웅 | ×1 | ×30 | ×3 |

**등급은 기관 선택 시 자동 결정된다** — 상세는 4장 참조.

### 2-4. 다중 루트 개방 (멀티-각성)

한 기본영웅이 하나의 기관으로 진화한 후에도, **다른 기관 루트를 추가로 개방**할 수 있다.

```
예시:
플레이어가 '아이리스'를 올림푸스로 진화 → "번개의 아이리스" (SSR) 획득
이후 발할라 루트 추가 개방 → "폭풍의 아이리스" (SR) 별도 획득

결과: 플레이어가 두 버전의 아이리스를 동시에 보유 가능
      파티에는 하나만 편성 가능 (같은 기본영웅 중복 편성 불가)
```

**추가 루트 개방 비용**: 첫 진화 비용의 1.5배 (기관 에센스 기준)

## 3. MVP 기본영웅 목록

### 설계 원칙

1. **이름 다양성**: 성별/국적/어감 혼합 (한국, 서구, 중동, 동양 이름 혼합)
2. **정체성 명확성**: 기본영웅 단계에서 "이 캐릭터는 어떤 사람인가"가 분명해야 함
3. **진화 호기심 유발**: "이 캐릭터를 어디로 보내야 하나" 고민이 즐거워야 함
4. **기관 분포**: 12개 기관이 MVP 내에서 골고루 분산 (한 기관에 집중 금지)

## 4. 전직 영웅 등급 체계

### 4-1. 등급 결정 방식 결정

**채택 방향: "기관 고유 등급" + "기본영웅과의 궁합 보정" 혼합**

#### 이유

순수 랜덤 방식 (기관×기본영웅 조합마다 등급이 달라지는 방식)은 관리 복잡도가 폭증한다. 10명 × 평균 2.5루트 = 25개 전직 영웅의 등급을 개별 설정해야 한다. 반면 "기관별 고정 등급"만 사용하면 전략의 다양성이 줄어든다.

따라서 **기본 원칙은 기관 등급으로 고정하되, 특정 기본영웅과의 궁합이 완벽하게 맞는 경우 등급을 1단계 상향하는 "공명 보정"을 적용**한다. 이는 수집 욕구와 의외성을 동시에 충족한다.

#### 기관 기본 등급

## 5. 재화 시스템

### 5-1. 재화 전체 구조

v2.0에서는 "미정령 → 기본영웅 → 전직 영웅"의 3단계 흐름에 맞춰 재화도 3레이어로 구성한다.

| 레이어 | 목적 | 재화 |
|--------|------|------|
| **레이어 1** | 기본영웅 획득 | 미정령 조각, 정령석 |
| **레이어 2** | 기관 진화 (전직) | 기관 각인서, 기관 에센스 (12종), 각성의 불꽃 |
| **레이어 3** | 추가 루트 개방 | 세계수의 씨앗 |

### 5-2. 재화 상세

#### 레이어 1 — 기본영웅 획득

| 재화명 | 설명 | 주요 획득처 |
|--------|------|-----------|
| **미정령 조각** | 각 기본영웅별 30개 수집 시 해당 기본영웅 획득 | 일반 배너 가챠, 스테이지 드롭, 이벤트 |
| **정령석** | 기본영웅 각성 확정 비용 | 일일 퀘스트, 전투 클리어, 주간 보상 |
| **범용 조각권** | 임의 기본영웅 조각 10개 → 원하는 기본영웅 조각으로 교환 | 월간 패스, 이벤트 클리어 |

#### 레이어 2 — 기관 진화 (전직)

| 재화명 | 설명 | 주요 획득처 |
|--------|------|-----------|
| **기관 각인서** | 진화 1회 시작권 (기관 무관 공통 사용) | 프리미엄 배너, 탑 클리어, 월간 패스 |
| **기관 에센스 (12종)** | 기관별 전용 진화 재료 | 해당 기관 테마 스테이지, 주간 레이드, 이벤트 |
| **각성의 불꽃** | SSR 등급 진화에만 필요 (공명 보정 포함) | 고난이도 탑, 시즌패스, 프리미엄 교환소 |

#### 레이어 3 — 추가 루트 개방

| 재화명 | 설명 | 주요 획득처 |
|--------|------|-----------|
| **세계수의 씨앗** | 이미 전직한 기본영웅의 다른 기관 루트 추가 개방 | 극한 난이도 클리어, 시즌 종료 보상, 결제 |

### 5-3. 재화 흐름 시뮬레이션 (일반 플레이어 기준)

#### 일일 수급량

| 재화 | 일일 수급량 |
|------|-----------|
| 미정령 조각 (모든 영웅 합산) | 8~15개 |
| 정령석 | 5~8개 |
| 기관 에센스 (특정 교단) | 10~15개 |
| 기관 각인서 | 0~1개 (주당 3~5개) |
| 각성의 불꽃 | 0~0.3개 (월 5~8개) |

#### 기본영웅 1명 획득까지 소요 기간

```
미정령 조각 30개 필요
일일 수급: 특정 기본영웅 조각은 배너 확률 기반
10연 가챠 시 조각 평균 3~5개 드롭 → 6~10회 10연 필요
배너 없이 교환소 활용 시: 범용 조각권 축적 → 약 2~3주
```

#### SSR 전직 영웅 1명 완성까지 소요 기간

```
기관 각인서 ×1 → 주당 3~5개 수급, 1~2일 분량
기관 에센스 ×30 → 일일 10~15개 수급, 약 2~3일 분량
각성의 불꽃 ×3 → 월 5~8개 수급, 약 2주 분량

합산: 기본영웅 보유 후 SSR 전직까지 약 2~3주 (무과금 기준)
      각성의 불꽃 구매 시: 즉시 가능 (과금 가속 포인트)
```

### 5-4. v1.0과 재화 체계 비교

| 항목 | v1.0 | v2.0 |
|------|------|------|
| 기본영웅 획득 | 미정령 가챠 (랜덤) | 조각 30개 + 정령석 (확정) |
| 전직 재화 | 교단 코인 (교단별) | 기관 각인서 + 기관 에센스 |
| SSR 확정 재화 | 신성의 결정 | 각성의 불꽃 |
| 다중 루트 | 없음 | 세계수의 씨앗 |
| 플레이어 자율성 | 낮음 (랜덤 결과) | 높음 (원하는 루트 선택) |

## 6. 게임 루프

### 6-1. 코어 루프 (일일)

```
전투 출격 (스테이지 / 탑 / 레이드)
    ↓
미정령 조각 + 기관 에센스 + 정령석 획득
    ↓
기본영웅 조각 30개 달성 → 기본영웅 각성
    ↓
기관 선택 고민 ("어떤 기관으로 키울까?")
    ↓
기관 각인서 + 에센스 + 각성의 불꽃 수급
    ↓
전직 영웅 탄생 → 파티 강화
    ↓
더 어려운 콘텐츠 도전 (반복)
```

### 6-2. 메타 루프

| 루프 | 주기 | 목표 |
|------|------|------|
| **일일 루프** | 매일 | 전투 → 재화 수급 → 조각 누적 |
| **주간 루프** | 주 1~2회 | 기관 에센스 레이드 → 전직 1~2회 |
| **중기 루프** | 2~4주 | 기본영웅 완성 → SSR 전직 → 파티 완성 |
| **장기 루프** | 월 단위 | 추가 루트 개방 → 컬렉션 완성 → 파티 최적화 |

### 6-3. 의사결정 트리 (플레이어 경험)

```
기본영웅 '아이리스' 획득
    │
    ├── [올림푸스 루트] → "번개의 아이리스" (SSR 폭딜)
    │       파티에 딜러가 부족한 경우 선택
    │
    ├── [발할라 루트] → "폭풍의 아이리스" (SSR 역전)
    │       파티에 탱커+역전 조합이 필요한 경우 선택
    │
    └── [카오스 루트] → "혼돈의 아이리스" (SSR 고분산)
            도박형 플레이를 즐기거나
            이미 올림푸스/발할라 버전을 보유한 경우 수집 목적

    → 세 가지 버전 모두 수집 시 "아이리스 컬렉션 완성" 달성
    → 특수 칭호 + 보너스 스탯 해금
```

### 6-4. 기존 교단 시너지와의 연계

전직 영웅이 특정 기관(교단)에 속하므로, 기존 `CULT_SYSTEM_DESIGN.md`의 파티 시너지 시스템과 완전히 호환된다.

```
예시 파티:
  - 번개의 아이리스 (olympus, warrior)
  - 신벌의 레온 (olympus, warrior)
  - [olympus SSR 힐러]

→ olympus 3명 시너지 발동:
  Lightning Strike 스턴 확률 50%→70%
  → 강력한 올림푸스 모노 파티 구성 가능
```

같은 기본영웅을 다른 루트로 키워 **다른 기관 시너지를 채우는** 전략도 가능하다.

**작성 문서**: `docs/EVOLUTION_SYSTEM_GDD.md`

**주요 결정사항**:

1. **기본영웅 정체성 우선 원칙**: 기존 v1.0의 랜덤 가챠 기반 전직 대신, 이름과 배경을 가진 기본영웅 10명을 먼저 정의하고 기관 선택으로 전직 방향이 확정된다. 플레이어 자율성과 캐릭터 애착이 핵심.

2. **등급 결정: 기관 기본등급 + 공명 보정**: 기관마다 SSR/SR/R 기본 등급을 부여하되, 특정 기본영웅과 설정상 완벽한 궁합(공명)이 있을 경우 1단계 상향된다. MVP 24개 전직 영웅 중 SSR 14개, SR 7개, R 3개로 설계되었다.

3. **멀티-각성 수집 시스템**: 한 기본영웅을 여러 기관 루트로 키울 수 있으며, 모든 루트 완성 시 컬렉션 달성 보너스를 제공한다. 장기 플레이 동기와 수집 욕구를 동시에 자극하는 구조.
