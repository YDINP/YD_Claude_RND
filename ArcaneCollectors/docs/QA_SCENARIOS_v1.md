# QA 시나리오 — BUGFIX_PRD_v1 수정 후 검증

## 테스트 환경
- **URL**: https://arcane-collectors.vercel.app
- **브라우저**: Chrome 모바일 에뮬레이션 (360×640) + 실제 모바일
- **기준**: BUGFIX_PRD_v1.md 14개 이슈 전부 수정 완료

---

## Phase 1: Critical

### TC-01: 팝업 열기/닫기 반복 (BUG-01)
1. 메인화면에서 영웅 리스트 팝업 열기
2. 팝업 닫기
3. 다시 열기 → **타이틀 "내 영웅"이 1개만 표시되는지 확인**
4. 5회 반복 → 메모리 누수/중복 없는지 확인

### TC-02: 재접속 시 보스전 준비 (BUG-12)
1. 게스트 로그인 → 메인화면
2. 방치 진행도가 100% 될 때까지 대기 (또는 치트패널에서 조정)
3. 브라우저 새로고침
4. **재접속 후 즉시 "보스전 준비!" 텍스트 + 버튼 활성화 확인**

### TC-03: 무한탑 진행도 (BUG-08)
1. 무한탑 진입 → 전투 → 클리어
2. **클리어 후 진행도 0%에서 시작 (다음 층) 확인**
3. 중도 패배 시 진행도 유지 확인

### TC-04: 초기 재화 (BUG-05)
1. localStorage 삭제 → 새 게임 시작
2. **보석 2700, 소환권 10장 확인**
3. 가차 1회 소환(보석 300) → 보석 2400 확인

---

## Phase 2: Depth/Layout

### TC-05: 메인화면 요소 겹침 (BUG-02/03/04/11)
1. 메인화면 로드
2. **TopBar가 다른 모든 요소 위에 표시되는지 확인**
3. **하단 메뉴 버튼(홈/영웅/소환/더보기) 터치 가능 확인**
4. **보상받기 버튼이 하단 메뉴와 겹치지 않고 독립 터치 확인**
5. **소탕/보스전 버튼이 모험 패널 위에서 정상 터치 확인**
6. **파티 패널, 전투력 패널, 모험 패널 순서대로 정상 렌더링 확인**

### TC-06: 챕터 정보 표시 (BUG-11)
1. 메인화면 모험 패널에서 **챕터명/스테이지 텍스트 가독성 확인**
2. 소탕/보스전 버튼 아래로 밀리지 않는지 확인

---

## Phase 3: UI Popup Overhaul

### TC-07: 영웅 리스트 그리드 (BUG-09)
1. 하단 메뉴 "영웅" 탭 클릭
2. **3열 그리드로 영웅 카드 표시 확인**
3. 영웅 10명 이상일 때 **스크롤 동작 확인 (마스크 영역 내)**
4. 등급 필터 (전체/N/R/SR/SSR) 동작 확인
5. 정렬 (레벨/등급/전투력) 동작 확인

### TC-08: 파티 편집 팝업 (BUG-06)
1. 메인화면 파티 패널 "편성" 버튼 클릭
2. **영웅 선택 영역이 3열 그리드로 표시되는지 확인**
3. 영웅 선택 → 파티 슬롯에 추가 확인
4. **편성 완료 후 메인화면 파티 영웅 갱신 확인**

### TC-09: 영웅 상세 팝업 (BUG-07 + BUG-14)
1. 영웅 리스트에서 영웅 카드 클릭
2. **스탯 정보 (HP/ATK/DEF/SPD) 표시 확인**
3. **스킬 목록 표시 확인**
4. **등급별 배경색 차이 확인**: N(회색), R(파랑), SR(보라), SSR(금색)
5. SSR 영웅과 N 영웅의 배경/테두리 색상 뚜렷히 다른지 확인

---

## Phase 4: Balance & Settings

### TC-10: 개발자 모드 토글 (BUG-10)
1. 하단 "더보기" → 설정 팝업
2. **"개발자 모드" 토글 표시 확인**
3. 토글 ON → **디버그 FAB 버튼 화면 우하단에 표시**
4. FAB 클릭 → 디버그 패널 열림
5. 토글 OFF → FAB 사라짐

### TC-11: 초반 보스 밸런스 (BUG-13)
1. 새 게임 시작 (전투력 ~400 기준)
2. 방치 진행 → **진행도 바가 체감 가능한 속도로 차오르는지 확인**
3. DPS 100 기준, 1-5 보스 HP ~300 → **약 3초면 진행도 100%**
4. 보스전 진입 → 클리어 가능한 난이도인지 확인

## 회귀 테스트

### TC-12: 씬 전환 안정성
1. 메인 → 스테이지 선택 → 뒤로가기 → 메인
2. 메인 → 가차 → 10연차 → 결과 확인 → 닫기 → 메인
3. 메인 → 무한탑 → 전투 → 결과 → 메인
4. **각 전환 시 콘솔 에러 없는지 확인**

### TC-13: 저장/로드
1. 플레이 후 브라우저 닫기
2. 재접속 → **재화, 파티, 진행도 모두 유지 확인**
3. 다른 디바이스에서 게스트 로그인 → 로컬 저장만이므로 별도 확인

### TC-14: 모바일 반응형
1. 360×640 (소형) → UI 잘림 없는지
2. 390×844 (iPhone 14) → 정상 표시
3. 768×1024 (태블릿) → 스케일링 정상
