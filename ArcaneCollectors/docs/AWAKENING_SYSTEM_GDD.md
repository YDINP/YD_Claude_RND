## 개요

캐릭터는 처음부터 교단에 소속되지 않는다. 방랑자(Wanderer) 상태로 소환되어, 플레이어와의 여정을 통해 특정 교단과 "공명(Resonance)"을 이루고 각성한다. 교단 각성은 단순한 능력치 강화가 아닌, 캐릭터의 정체성이 완성되는 서사적 사건이다.

---

## 1. 미각성 상태 — 방랑자(Wanderer)

### 정의

- 모든 캐릭터는 소환 직후 **방랑자** 상태로 시작
- 교단 시너지에 참여 불가 (파티 구성 시 보너스 미적용)
- 전용 스킬 대신 **범용 스킬** 세트 보유
- 카드 UI: 흑백 실루엣 + "방랑자" 뱃지
- 스탯: 기본 스탯만 적용, 교단 보너스 없음

### 방랑자 범용 스킬

| 스킬 | 효과 |
|------|------|
| 기본 공격 | 단일 적 ATK×1.0 피해 |
| 방어 자세 | 1턴 DEF +20% |
| 회복 | 자신 HP 10% 회복 |

## 2. 각성 조건 (3가지 필수)

### 조건 A: 친밀도 (Affinity)

- 교단별 친밀도 요구 수준 상이
- 전투에서 함께 싸우거나, 교단 관련 스테이지 클리어 시 친밀도 상승

| 교단 등급 | 요구 친밀도 레벨 |
|----------|----------------|
| SSR (prism_stars, neon_crow, stella_club, lunatic_circus) | Lv. 30 |
| SR (ink_cyclone, card_cartel, buddy_garden, glitch_paradise, iron_beat) | Lv. 25 |
| R (cafe_encore) | Lv. 20 |

### 조건 B: 교단 전용 소재 (Cult Material)

각 교단별 고유 소재 수집 필요:

| 교단 | 소재명 | 요구량 |
|------|--------|--------|
| Prism Stars | 공명의 별빛 조각 | 30개 |
| Neon Crow | 암호화된 데이터 코어 | 30개 |
| Ink Cyclone | 잉크 정수 | 25개 |
| Stella Club | 별자리 파편 | 30개 |
| Card Cartel | 봉인된 황금 카드 | 25개 |
| Buddy Garden | 공생의 씨앗 | 25개 |
| Glitch Paradise | 카오스 오류 데이터 | 25개 |
| Cafe Encore | 중립의 커피 원두 | 20개 |
| Lunatic Circus | 환상의 마술 인장 | 30개 |
| Iron Beat | 언더그라운드 공연 기록 | 25개 |

**소재 획득처**: 교단 관련 스테이지 드랍, 이벤트 던전, 교단 시련 보상

### 조건 C: 교단 시련 (Cult Trial)

교단별 고유 챌린지 스테이지 클리어:

| 교단 등급 | 요구 시련 등급 |
|----------|--------------|
| SSR | 시련 3성 클리어 |
| SR | 시련 2성 클리어 |
| R | 시련 1성 클리어 |

**시련 특징**:
- 해당 교단 보너스 스테이지 (시너지 보너스 없이 도전)
- 방랑자 상태에서도 도전 가능 (사전 체험 역할)

### 조건 D: SSR 추가 조건 — 교단 각인석

SSR 등급 교단(prism_stars, neon_crow, stella_club, lunatic_circus) 각성 시 추가 필요:

| 항목 | 내용 |
|------|------|
| 아이템명 | 교단 각인석 (Cult Imprint Stone) |
| 요구량 | 1개 |
| 무료 획득 | 월 1회 무료 지급 (우편함) |
| 유료 획득 | 다이아 500개 교환 |

## 3. 각성 진행도 5단계

캐릭터는 조건 충족 정도에 따라 단계적으로 교단과 가까워진다.

```
방랑자 → 관심 → 입문 → 수련 → 서약 → [각성 완료]
```

| 단계 | 이름 | 조건 달성률 | 효과 |
|------|------|-----------|------|
| 0 | 방랑자 (Wanderer) | 0% | 범용 스킬, 교단 무소속 |
| 1 | 관심 (Interest) | 친밀도 50% | 교단 소개 대화 해금, 카드 컬러 힌트 |
| 2 | 입문 (Initiation) | 친밀도 100% | 교단 아이덴티티 스킬 1개 선공개 |
| 3 | 수련 (Training) | 소재 50% | 기본 시너지 참여 가능 (50% 효율) |
| 4 | 서약 (Oath) | 소재 100% + 시련 완료 | 전용 스킬 세트 잠금 해제 |
| 5 | 각성 완료 | 모든 조건 충족 | 교단 정식 가입, 풀 시너지, 각성 카드 UI |

## 4. 각성 연출 (Awakening Cutscene)

### 공통 플로우

1. **프리롤** (1~2초): 화면 전체 검게 페이드 아웃
2. **실루엣 크랙** (2~3초): 흑백 실루엣에 교단 컬러 균열 발생
3. **교단 상징 등장** (2~3초): 교단 엠블럼이 배경 전면에 전개
4. **각성 선언** (1초): 캐릭터 대사 + 교단 이름 텍스트 이펙트
5. **풀컬러 공개** (2~3초): 교단 컬러로 물든 캐릭터 카드 최종 공개

**총 연출 시간**: 8~12초 (스킵 불가, 최초 1회만)

### 교단별 연출 차별화

## 5. 내러티브 — "공명(Resonance)" 개념

### 세계관 설명

이 세계에서 교단은 단순한 조직이 아닌, **살아있는 에너지 집단**이다. 각 교단은 고유한 "공명 주파수"를 가지며, 캐릭터가 충분한 경험과 감정을 쌓았을 때 해당 교단과 주파수가 일치하는 순간이 온다. 이것이 각성이다.

> "교단은 선택하는 것이 아니다. 공명이 일어나는 순간, 교단이 당신을 찾아온다."
> — 아케인 컬렉터즈 세계관 핵심 원칙

### 각 교단별 각성 트리거 (스토리 모멘트)

| 교단 | 각성 트리거 키워드 |
|------|------------------|
| Prism Stars | "처음으로 누군가의 앞에 서서 노래를 불렀을 때" |
| Neon Crow | "시스템의 거짓말을 처음으로 스스로 간파했을 때" |
| Ink Cyclone | "지울 수 없는 무언가를 세상에 남기고 싶었을 때" |
| Stella Club | "밤하늘을 보며 자신이 우주의 일부임을 느꼈을 때" |
| Card Cartel | "처음으로 한 수 앞을 내다보고 결과가 맞아떨어졌을 때" |
| Buddy Garden | "타인의 성장을 자신의 기쁨으로 느꼈을 때" |
| Glitch Paradise | "실수가 오히려 더 나은 결과를 만들었음을 깨달았을 때" |
| Cafe Encore | "적과 아군 없이 그냥 함께 쉬고 싶었을 때" |
| Lunatic Circus | "불가능한 것을 가능하게 만들어 보는 것 자체가 즐거웠을 때" |
| Iron Beat | "억눌렸던 무언가를 소리로 터뜨리고 싶었을 때" |

## 6. UI/UX 설계

### 캐릭터 카드 변화

| 상태 | 카드 디자인 |
|------|-----------|
| 방랑자 | 흑백 실루엣, 회색 프레임, "방랑자" 배지 |
| 관심 단계 | 프레임 컬러 힌트 시작, 캐릭터 눈 컬러 출현 |
| 입문~수련 | 교단 컬러 점진적 채색 (30% → 70%) |
| 서약 | 95% 채색, 교단 엠블럼 반투명 워터마크 |
| 각성 완료 | 풀컬러, 교단 프레임, 엠블럼 배지, 각성 이펙트 |

### 각성 진행도 UI

- 캐릭터 상세 화면 하단에 **교단 선택 슬롯** 표시
- 진행도 게이지 (5단계 세그먼트 바)
- 각 조건 체크리스트 (친밀도/소재/시련 달성률 표시)
- "각성 가능" 상태 시 카드에 황금 테두리 펄스 이펙트

### 교단 선택 시스템

- 방랑자는 **최대 2개 교단 후보** 동시 진행 가능
- 3번째 교단 진행 시 기존 1개 포기 선택 필요
- 각성 완료 후 다른 교단 진행 불가 (교단은 1캐릭터 1교단)

### 알림 & 피드백

| 이벤트 | 알림 유형 |
|--------|---------|
| 친밀도 단계 상승 | 화면 상단 토스트 + 대화 이벤트 |
| 소재 목표 달성 | 카드 뱃지 업데이트 |
| 시련 클리어 | 팝업 + 진행도 게이지 애니메이션 |
| 각성 가능 | 메인 화면 배너 + 푸시 알림 |
| 각성 완료 | 풀스크린 컷신 (8~12초) |

## 7. 경제 설계

### 소재 드랍율

| 획득처 | 드랍 소재 | 평균 획득량 |
|--------|---------|-----------|
| 교단 관련 스테이지 1회 | 해당 교단 소재 | 1~3개 |
| 이벤트 던전 (교단 테마) | 교단 소재 × 1.5 | 2~5개 |
| 교단 시련 1성 | 교단 소재 보너스 | 10개 |
| 교단 시련 2성 | 교단 소재 보너스 | 20개 |
| 교단 시련 3성 | 교단 소재 보너스 | 30개 |

### 예상 플레이 타임 (SR 교단 기준)

| 단계 | 예상 소요 일수 (일일 퀘스트 수행 기준) |
|------|--------------------------------------|
| 방랑자 → 관심 | 2~3일 |
| 관심 → 입문 | 3~5일 |
| 입문 → 수련 | 5~7일 |
| 수련 → 서약 | 7~10일 |
| 서약 → 각성 완료 | 3~5일 (시련 포함) |
| **전체 SR 각성** | **20~30일** |

SSR 교단은 각인석 추가로 +7~14일 예상.

## 8. 밸런싱 고려사항

### 각성 vs 미각성 스탯 차이

- 각성 후 교단 시너지 보너스 풀 적용
- 방랑자 상태에서도 개인 스탯은 레벨업으로 성장 가능
- 각성이 필수는 아니지만, 파티 시너지 위해 권장

### 다중 각성 방지

- 1캐릭터당 1교단 최종 각성 (중간 포기 가능, 재도전 불가)
- 교단 각인석을 신중하게 사용하도록 유도

### 과금 압박 방지

- 월 1회 무료 각인석으로 SSR 교단 1회 각성 보장
- 무료 플레이어도 30일이면 SSR 1개 각성 가능

## 10. 파일 구현 계획

```
src/
├── systems/
│   └── AwakeningSystem.js    ← 각성 조건 체크, 진행도 관리
├── data/
│   ├── cults.json            ← v3.0 (10개 서브컬쳐 교단, 각성 조건 포함)
│   └── cult-materials.json   ← 교단별 소재 정의
└── components/
    └── popups/
        └── AwakeningPopup.js ← 각성 진행도 UI + 조건 체크리스트
```

### AwakeningSystem.js 핵심 메서드

```javascript
// 각성 진행도 조회
getAwakeningProgress(heroId, cultId) → { stage, affinity, materials, trialCleared }

// 각 조건 달성 여부
checkAffinityCondition(heroId, cultId) → boolean
checkMaterialCondition(heroId, cultId) → { current, required, fulfilled }
checkTrialCondition(heroId, cultId) → boolean

// 각성 실행
executeAwakening(heroId, cultId) → { success, error? }

// 방랑자 여부
isWanderer(heroId) → boolean
```
