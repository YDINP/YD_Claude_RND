## 1. 시스템 개요

### 한 줄 컨셉
> 미정령에서 전설적 영웅으로 — 교단의 선택이 운명을 결정한다.

### 핵심 경험 목표 (Experience Pillars)
1. **수집의 재미**: 어떤 영웅이 나올지 모르는 랜덤 전직의 설렘
2. **전략적 선택**: 교단 지정과 클래스 조합을 통한 전략적 파티 구성
3. **장기 투자감**: 천장(확정 전직) 목표를 향한 지속적인 플레이 동기

## 2. 캐릭터 3-티어 구조

### 티어 개요

```
[미정령 — Unbound Spirit]
  형태 불명, 어느 교단에도 속하지 않은 원초적 존재
  기본 전투/파티 등록 가능 (기본 캐릭터)
  최대 레벨 20, 능력치 40~50% 발현
  → 정령석 ×5 소모 → 교단 지정

[일반캐릭터 — Common Hero]
  교단 지정 완료, 클래스 각성
  기본 전투/파티 등록 가능 (추천 라인업)
  최대 레벨 40, 능력치 60~70% 발현
  → 교단 코인 소모 → 전직

[전직 영웅 — Ascended Hero]
  SSR: 최대 레벨 60, 100% 능력치 + 전용 스킬
  SR:  최대 레벨 50, 85% 능력치
  R:   최대 레벨 40, 70% 능력치
```

### 미정령 / 일반캐릭터의 역할
- **기본 파티 구성 가능**: 미정령/일반캐릭터만으로도 전투 및 파티 등록 가능
- **추천 시스템**: 교단/클래스 조합 기반 최적 전직 영웅 추천 UI
- **전직 대기 캐릭터**: 플레이어가 전직 재화를 모으는 동안 실전 투입 가능

## 3. 전직 시스템 상세

### 3-1. 교단 지정 (미정령 → 일반캐릭터)

| 항목 | 내용 |
|------|------|
| 소모 재화 | 정령석 ×5 |
| 교단 선택 | 12개 교단 중 1개 지정 |
| 교단 변경 | 신성의 결정 ×10 (언제든 가능) |
| 클래스 결정 | 교단 지정 시 클래스도 함께 선택 |

### 3-2. 전직 방법 (일반캐릭터 → 전직 영웅)

#### 랜덤 전직 (Regular Ascension)
| 방법 | 소모 재화 | 확률 | 비고 |
|------|---------|------|------|
| 단일 전직 | 교단 코인 ×10 | R 82% / SR 15% / SSR 3% | 해당 교단×클래스 풀에서 랜덤 |
| 10연 전직 | 교단 코인 ×90 | SR 이상 1체 보장 | 10% 할인 + 보장 |

#### 확정 전직 / 천장 (Confirmed Ascension)
| 방법 | 소모 재화 | 결과 |
|------|---------|------|
| 원하는 영웅 확정 | 교단 코인 ×50 + 신성의 결정 ×5 | 지정 영웅 1체 확정 획득 |

### 3-3. 피티 (Pity) 시스템

```
소프트 피티: 30회 시작 → 매 회마다 SSR 확률 +1%
             (30회: 3% → 31회: 4% → ... → 50회: 23% + 천장)

하드 피티: 50회 SSR 100% 확정
           (계정 전체 아닌 교단×클래스 조합별 독립 카운트)

SR 미니 피티: 10회마다 SR 이상 1체 보장 (10연 기준)
```

## 4. 재화 시스템

### 재화 종류

| 재화명 | 한국어명 | 획득 방법 | 소모처 |
|--------|---------|---------|-------|
| Spirit Stone | 정령석 | 전투 클리어, 일일 퀘스트 | 교단 지정 (미정령 → 일반캐릭터) |
| Cult Coin | 교단 코인 | 전투 보상, 이벤트, 교단 활동 | 랜덤 전직, 확정 전직 |
| Divine Crystal | 신성의 결정 | 결제, 시즌패스, 극한 난이도 | 확정 전직, 교단 변경 |

### 일일 재화 수급 목표 (일반 플레이어 기준)

| 재화 | 일일 수급량 | 일일 소모 적정량 | 목표 비율 |
|------|-----------|--------------|---------|
| 정령석 | 8~12개 | 5개 (1회 지정) | 흑자 유지 |
| 교단 코인 | 15~25개 | 10개 (1회 전직) | 2~3일 1회 전직 |
| 신성의 결정 | 0~2개 | - | 과금/이벤트 중심 |

## 5. 추천 시스템

### UI 흐름
```
일반캐릭터 보유 시 → [전직 추천] 뱃지 표시
→ 클릭 시: 보유 교단 코인 / 해당 교단 영웅 목록 / 추천 영웅 표시
→ 추천 기준: 현재 파티의 약점 보완 영웅
```

### 추천 알고리즘 (기획 단계)
1. 파티 내 없는 클래스 분석
2. 해당 교단 × 누락 클래스 조합 우선 추천
3. 이미 보유한 영웅 제외
4. SSR → SR → R 순으로 추천 정렬

## 6. 게임 루프 통합

```
미정령 획득 (전투 보상)
    ↓ 정령석 ×5
교단 + 클래스 지정 (일반캐릭터)
    ↓ 실전 투입 가능 (추천 라인업)
교단 코인 ×10~90 소모
    ↓ 랜덤 전직 (설렘)
전직 영웅 획득 → 파티 강화
    ↓ 더 어려운 콘텐츠 도전
보상으로 재화 수급 → (반복)
```

### 메타 루프
```
일일: 전투 → 재화 수급 → 미정령/일반캐릭터 육성
주간: 충분한 교단 코인 → 다수 전직 시도
장기: 피티 카운트 쌓기 → 천장/확정 전직 → 파티 완성
```

## 7. MVP 범위 (Must Have)

### 목표 캐릭터 구성
| 항목 | 목표치 |
|------|--------|
| 교단 수 (MVP) | 4개 (olympus / valhalla / kunlun / avalon) |
| 클래스 수 | 3개 (warrior / mage / healer) |
| 등급 | SSR / SR / R |
| **MVP 영웅 총 수** | **4교단 × 3클래스 × 3등급 = 36명** |

### Must Have
- [ ] 미정령 → 일반캐릭터 교단 지정 UI
- [ ] 랜덤 전직 (단일/10연)
- [ ] 피티 카운트 저장 및 표시
- [ ] 전직 영웅 데이터 (MVP 36명)
- [ ] 추천 시스템 기본 UI

### Should Have
- [ ] 확정 전직 (천장)
- [ ] 교단 변경 기능
- [ ] 추천 알고리즘 (파티 약점 분석)

### Could Have
- [ ] 8교단 이상 확장
- [ ] 전직 연출 애니메이션
- [ ] 영웅 도감 / 컬렉션
